# 06-08 中考英语语法填空 练习二

主题：垃圾分类 (Rubbish Sorting)

## 一、题目

阅读下面短文，在空白处填入一个适当的单词，或填入括号中所给单词的正确形式。

Rubbish sorting has become a hot topic in China. Since 2019, many cities ___1___ (introduce) strict rules about how to sort household waste. In Shanghai, for example, rubbish must ___2___ (divide) into four categories: recyclable waste, harmful waste, wet waste and dry waste. Breaking the rules can result ___3___ a fine. At first, many residents found it ___4___ (confuse) because they didn't know which bin to use. Some communities set up smart bins that can ___5___ (automatic) identify the type of rubbish. Our school also started a sorting programme last September. Each classroom now has four ___6___ (colour) bins. Students take turns ___7___ (check) whether the rubbish is sorted correctly. The programme has been very ___8___ (success) so far. Not only has the amount of waste ___9___ (reduce), but students have also become more aware of environmental protection. I believe that if everyone does ___10___ (they) part, we can build a cleaner world.

---

## 二、答案与解析

1. **答案：have introduced**
   - **考点**：动词时态。since 2019 提示用现在完成时，主语 many cities 为复数，填 have introduced。

2. **答案：be divided**
   - **考点**：动词语态。rubbish 与 divide 之间为被动关系，must 后接动词原形的被动形式，填 be divided。

3. **答案：in**
   - **考点**：介词。result in 表示"导致"，固定搭配，填 in。

4. **答案：confusing**
   - **考点**：词性转换。find it confusing 表示"觉得令人困惑"，形容物的感受用 -ing 形式，confuse → confusing。

5. **答案：automatically**
   - **考点**：词性转换。修饰动词 identify，需用副词形式，automatic → automatically。

6. **答案：colourful/coloured**
   - **考点**：词性转换。修饰名词 bins，用形容词形式，colour → colourful（色彩丰富的）或 coloured（有颜色标记的）。

7. **答案：checking**
   - **考点**：非谓语动词。take turns doing sth. 表示"轮流做某事"，填 checking。

8. **答案：successful**
   - **考点**：词性转换。has been 后接形容词作表语，success → successful。

9. **答案：been reduced**
   - **考点**：动词时态与语态。the amount of waste 与 reduce 之间为被动关系，has 已给出，后接 been reduced 构成现在完成时的被动语态。

10. **答案：their**
    - **考点**：代词。do one's part 表示"尽自己的一份力"，everyone 在此用 their（泛指），填 their。
