# 代词——不定代词 some / any / no / every 及其复合词（Indefinite Pronouns — some / any / no / every and Their Compounds）

> 日期：2027 年 8 月 7 日 | 语法分类：词法 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**不定代词（indefinite pronouns）**指不具体指代某个人或事物的代词。英语中常用 **some、any、no、every** 与 **-one、-body、-thing、-where** 等后缀构成复合不定代词（compound indefinite pronouns），如 *someone、anything、nothing、everywhere*。它们既可表示人、物，也可表示地点，用法与 **some / any / no / every** 的基本语气一致：**some** 偏肯定或期待肯定回答；**any** 常用于疑问与否定；**no** 表全否定；**every** 表“全体中的每一个”。

### 2. 构成形式

| 前缀（prefix） | + one | + body | + thing | + where |
| :---: | :---: | :---: | :---: | :---: |
| **some** | someone | somebody | something | somewhere |
| **any** | anyone | anybody | anything | anywhere |
| **no** | no one | nobody | nothing | nowhere |
| **every** | everyone | everybody | everything | everywhere |

说明：**someone / somebody、anyone / anybody、no one / nobody、everyone / everybody** 意义相近，口语中 *-body* 更随意；**no one** 常分开写，**nobody** 更口语。

### 3. 用法与语境

- **some 及其复合词**：多用于**肯定句**；在表示**请求、建议或期待对方同意**的疑问句中也可用，如 *Would you like something to drink?*（对应精讲 1、本条）。
- **any 及其复合词**：多用于**否定句**和**一般疑问句**；在**条件句（if）**中表“任一”时也常用 *any-*（对应精讲 1、本条）。
- **no 及其复合词**：表示**否定**，本身已含否定意义，**不可再与 not 连用**构成双重否定错误（如不说 *I don't know nothing* 表“不知道”，应说 *I know nothing* 或 *I don't know anything*）（对应精讲 1、本条）。
- **every 及其复合词**：表示“**每一个 / 所有**”，强调整体中的个体；**everyone / everybody** 只指人，不指物（对应精讲 1、本条）。
- **主谓一致与修饰语**：复合不定代词作主语时，谓语动词一般用**单数**；形容词修饰时要**后置**，如 *something important*（对应本条）。

### 4. 标志词 / 关键信号

- 句中有 **not、never、hardly** 等否定词 → 常用 **any-** 或 **no-**（*no-* 已否定则动词不再加 not 表同一层否定）。
- **疑问句**且非“请求/建议”语气 → 常用 **any-**。
- **if / unless** 引导的条件句里“任何一个” → 常用 **any-**。
- **every-** 常与表范围的语言共现（如 *in our class、in the world*），表示全体中的每一个。

### 5. 易错点与辨析

1. **some- 与 any- 在疑问句**：普通疑问用 *any-*；期待肯定或表礼貌请求用 *some-*（易混点 1）。
2. **no- 与 not…any-**：*I have no friends.* = *I don't have any friends.*；避免 *no…not* 混用（易混点 2）。
3. **everyone 与 every one**：*everyone* 仅指“人人”；*every one of…* 可指人或物，*one* 可接 *of*（易混点 3）。
4. **形容词位置**：须说 *something nice*，不说 *nice something*（易混点 4）。
5. **作主语谓语用单数**：*Everyone is here.* 不用 *are*（易混点 5）。

### 逐题解析

**1. 答案：B**  
**解析**：肯定句表示“出了点问题”常用 *something wrong*。  
**排除法**：A *anything* 多用于疑问/否定；C 与后文要修理矛盾；D 不合逻辑。  
**考查知识点**：对应**三、用法与语境**第一条（some 及其复合词多用于肯定句）。

**2. 答案：D**  
**解析**：*can't* 表否定，否定句中指地点用 *anywhere*。  
**排除法**：A *somewhere* 多用于肯定；B 与 *can't find* 搭配会产生不自然双重否定感；C 与“找不到”语义不符。  
**考查知识点**：对应**三、用法与语境**第二条（any 及其复合词用于否定句）。

**3. 答案：A**  
**解析**：句意为“班里每个人都喜欢新老师”，用 *Everyone*。  
**排除法**：B、C 不表全体；D 与句意相反。  
**考查知识点**：对应**三、用法与语境**第四条（every 及其复合词表示“每个/所有”）。

**4. 答案：B**  
**解析**：*If you need any help* 为条件句中常见说法，*any* 表“任何一点”。  
**排除法**：A *some* 虽偶见于委婉条件句，但初中阶段与 *if* 搭配表“任一”时 *any* 更标准；C、D 与句意不符。  
**考查知识点**：对应**三、用法与语境**第二条（条件句中的 any）。

**5. 答案：C**  
**解析**：盒子是空的，用 *nothing*。  
**排除法**：A、B、D 均表示“有东西”，与 *empty* 矛盾。  
**考查知识点**：对应**三、用法与语境**第三条（no / nothing 表否定）。

**6. 答案：A**  
**解析**：*Would you like…?* 表委婉请求、期待对方肯定考虑，常用 *something*。  
**排除法**：B 多用于非请求型疑问；C、D 与请客/礼貌询问语境不符。  
**考查知识点**：对应**三、用法与语境**第一条与**五、易错点 1**（请求类疑问句用 some-）。

**7. 答案：B**  
**解析**：*don't* 否定，宾语用 *anything*。  
**排除法**：A 多用于肯定；C 与 *don't* 连用易构成不规范双重否定；D 与“还不知道”不符。  
**考查知识点**：对应**三、用法与语境**第二条（否定句用 any-）。

**8. 答案：C**  
**解析**：“到处都找了”用 *looked everywhere*。  
**排除法**：A、B 与“仍找不到包”相比，*everywhere* 更贴切表达找遍各处；D 语义不符。  
**考查知识点**：对应**三、用法与语境**第四条（everywhere 表所有地方）。

**9. 答案：A**  
**解析**：“一切都准备好了”用 *Everything is ready*。  
**排除法**：B、C、D 与“出发吧”的语境不匹配。  
**考查知识点**：对应**三、用法与语境**第四条（everything 表全部事物/情况）。

**10. 答案：B**  
**解析**：一般疑问句询问“有没有重要消息”，用 *anything*。  
**排除法**：A 多用于期待肯定的疑问；C、D 与询问事实的常见说法不符。  
**考查知识点**：对应**三、用法与语境**第二条与**五、易错点 1**（普通疑问用 any-）。

**11. 答案：C**  
**解析**：后文说仍不懂，说明他什么也没说，用 *nothing*。  
**排除法**：A、D 与“仍不理解”矛盾；B 多用于否定句/疑问句，此处缺否定词不宜单独用 *anything* 表“没说”。  
**考查知识点**：对应**三、用法与语境**第三条（nothing 表全否定）。

**12. 答案：C**  
**解析**：“全班都努力因而都及格”，主语用 *Everyone in our class*。  
**排除法**：A、D 与“都及格”相反；B 与全体事实不符。  
**考查知识点**：对应**三、用法与语境**第四条（everyone 指全体中的人）。

**13. 答案：A**  
**解析**：肯定句提议去“某个好玩的地方”，用 *somewhere*。  
**排除法**：B 多用于疑问/否定或“任意一处”的泛指；C、D 与愉快周末提议不符。  
**考查知识点**：对应**三、用法与语境**第一条（肯定句用 some-）。

**14. 答案：B**  
**解析**：*There isn't* 否定，修饰不可数名词 *milk* 用 *any*。  
**排除法**：A *some* 一般不用在否定 *isn't* 后表此义；C、D 结构不通。  
**考查知识点**：对应**三、用法与语境**第二条（否定句用 any）。

**15. 答案：D**  
**解析**：考场规则指向室内每一个人，用 *Everyone*。  
**排除法**：A、B 不表全体；C 与必须保持安静的要求相反。  
**考查知识点**：对应**三、用法与语境**第四条与**五、易错点 5**（everyone 作主语，语义为全体）。

**16. 答案：A**  
**解析**：形容词修饰复合不定代词须**后置**：*something important*。  
**排除法**：B、D 语序错误；C *anything* 用于此肯定陈述不自然。  
**考查知识点**：对应**三、用法与语境**第五条与**五、易错点 4**（形容词后置）。

**17. 答案：C**  
**解析**：*have nowhere to go* 表示“无处可去”，与待在家一致。  
**排除法**：A、B、D 与“只能待在家”因果不符。  
**考查知识点**：对应**三、用法与语境**第三条（nowhere 表否定地点）。

**18. 答案：D**  
**解析**：答语说全靠自己，即没人帮忙，用 *Nobody*。  
**排除法**：A 与 *by myself* 矛盾；B、C 与独自完成不符。  
**考查知识点**：对应**三、用法与语境**第三条（nobody / no one 表否定指人）。

**19. 答案：D**  
**解析**：“你喜欢坐哪都行”表任选位置，用 *anywhere*。  
**排除法**：A 表某一处而非任选；B、C 与自由就座规则不符。  
**考查知识点**：对应**三、用法与语境**第二条（anywhere 表任意地点）。

**20. 答案：D**  
**解析**：*Not every student* 表示“并非每个学生”，部分否定，为固定搭配。  
**排除法**：A *Not some* 不规范；B *Not any* 表全无，与 *but most do* 矛盾；C *Not no* 结构错误。  
**考查知识点**：对应**三、用法与语境**第四条与**五、易错点 3**（every 与部分否定 *not every*）。
