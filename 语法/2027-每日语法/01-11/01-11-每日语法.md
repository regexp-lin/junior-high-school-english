# 被动语态——含情态动词的被动语态（Passive Voice with Modal Verbs）

> 日期：2027 年 01 月 11 日 | 语法分类：语态 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

含情态动词的被动语态是中考常考的语法点。当句子中含有 can, could, may, might, must, should, will, would, need 等情态动词，且主语是动作的承受者时，就需要使用含情态动词的被动语态。

### 2. 构成形式

**核心结构：主语 + 情态动词 + be + 过去分词 (done)**

| 句型 | 结构 | 示例 |
|------|------|------|
| 肯定句 | 主语 + 情态动词 + be + done | Homework **must be finished** on time. |
| 否定句 | 主语 + 情态动词 + not + be + done | Photos **can't be taken** here. |
| 一般疑问句 | 情态动词 + 主语 + be + done? | **Should** the room **be cleaned** now? |
| 特殊疑问句 | 疑问词 + 情态动词 + 主语 + be + done? | **Where can** the book **be found**? |

**各情态动词的被动语态：**

| 情态动词 | 被动结构 | 含义 | 示例 |
|---------|---------|------|------|
| can/could | can/could be done | 能够被…… | The problem **can be solved**. |
| may/might | may/might be done | 可能被…… / 可以被…… | You **may be invited** to the party. |
| must | must be done | 必须被…… | Rules **must be followed**. |
| should | should be done | 应该被…… | The elderly **should be respected**. |
| will/would | will/would be done | 将会被…… | The task **will be completed** soon. |
| have to | have to be done | 不得不被…… | The plan **has to be changed**. |
| need | need to be done | 需要被…… | The car **needs to be repaired**. |

### 3. 用法与语境

- **用法一：can/could be done — 表示"能够被……"或"可以被……"**
  - 例：The question **can be answered** easily.（这个问题能够很容易地被回答。）
  - 例：The noise **could be heard** from far away.（噪音从远处都能被听到。）

- **用法二：must be done — 表示"必须被……"**
  - 例：Homework **must be handed in** before Friday.（作业必须在周五之前上交。）
  - 例：The rules **must be obeyed** by everyone.（规则必须被每个人遵守。）

- **用法三：should be done — 表示"应该被……"**
  - 例：The old people **should be taken care of**.（老人应该被照顾。）
  - 例：More trees **should be planted** to protect the environment.（应该种更多的树来保护环境。）

- **用法四：may be done — 表示"可能被……"或"被允许……"**
  - 例：The match **may be put off** because of the rain.（比赛可能因为下雨被推迟。）
  - 例：Students **may not be allowed** to leave early.（学生可能不被允许早退。）

### 4. 标志词 / 关键信号

| 信号 | 说明 | 示例 |
|------|------|------|
| 主语是动作承受者 + 情态动词 | 需要用情态被动 | The problem **must** be solved. (problem 是被解决的) |
| 题干出现 can't / mustn't / shouldn't | 情态否定被动 | Photos **can't be taken** here. |
| by + 执行者（可省略） | 提示被动语态 | It **should be done** by everyone. |
| 含有"被……""应该被……""能够被……"的中文语义 | 翻译提示 | 这些花应该被浇水。→ ...should be watered. |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | The room must cleaned every day. | The room must **be** cleaned every day. | 情态动词 + **be** + done，不能省略 be |
| 2 | The work can finished soon. | The work can **be finished** soon. | 同上，must/can/should 后必须有 be |
| 3 | Should the window be open? | Should the window be **opened**? | opened 是过去分词（被打开）；open 在此是形容词（开着的），语义不同 |
| 4 | The old must be take care of. | The old must be **taken** care of. | take 的过去分词是 taken |
| 5 | He can be saw in the library. | He can be **seen** in the library. | see 的过去分词是 seen，不是 saw |

### 逐题解析

**第 1 题：答案 B**

- **解析**：This problem 是被解决的（承受者），用情态被动 can be solved。
- **排除法**：A 为主动语态；C 缺少 be；D solve 不是过去分词。
- **考查知识点**：用法一——can be done 表示"能够被……"

**第 2 题：答案 C**

- **解析**：Homework 是被完成的，用 must be finished。
- **排除法**：A 为主动；B finish 不是过去分词；D 缺少 be。
- **考查知识点**：用法二——must be done 表示"必须被……"

**第 3 题：答案 C**

- **解析**：meeting 是被推迟的，Should the meeting be put off? 是含 should 的被动疑问句。
- **排除法**：A 缺少 be；B Must 语气太强且已有回答确认；D 缺少 be。
- **考查知识点**：构成形式——情态被动的一般疑问句

**第 4 题：答案 B**

- **解析**：The old people 是被照顾的，should be taken care of，take 的过去分词是 taken。
- **排除法**：A 为主动语态；C took 不是过去分词；D 缺少 be。
- **考查知识点**：用法三——should be done；动词短语被动（take care of）

**第 5 题：答案 C**

- **解析**：Photos 是被拍的，不被允许拍照用 can't be taken。take 的过去分词是 taken。
- **排除法**：A/B 为主动语态；D took 不是过去分词（应为 taken）。
- **考查知识点**：构成形式——can't be done 表示"不能被……"

**第 6 题：答案 B**

- **解析**：Trees 是被种植的，should be planted 表示"应该被种植"。
- **排除法**：A 为主动；C 缺少 be；D plant 不是过去分词。
- **考查知识点**：用法三——should be done

**第 7 题：答案 C**

- **解析**：The door 是被关的，must be closed 表示"必须被关上"。
- **排除法**：A 为主动；B close 是形容词而非过去分词；D 缺少 be。
- **考查知识点**：用法二——must be done；易错点——close(形容词) vs closed(过去分词)

**第 8 题：答案 B**

- **解析**：The food 是被保存的，need 作实义动词用 needs to be kept。
- **排除法**：A 缺少 to be；C 为主动（keep 应为 be kept）；D 缺少 to。
- **考查知识点**：need to be done 的结构

**第 9 题：答案 B**

- **解析**：Students 是被允许/不被允许的，shouldn't be allowed 表示"不应该被允许"。
- **排除法**：A 为主动；C 为主动且语义相反；D allow 不是过去分词。
- **考查知识点**：用法三——shouldn't be done 表否定建议

**第 10 题：答案 B**

- **解析**：this work 是被完成的，Can + 主语 + be + done 是情态被动疑问句，be finished。
- **排除法**：A 为原形主动；C finish 不是过去分词；D 缺少 be。
- **考查知识点**：构成形式——情态被动一般疑问句

**第 11 题：答案 B**

- **解析**：Children 是被允许/不被允许的，mustn't be allowed 表示"绝对不允许"。
- **排除法**：A 为主动；C 语义相反（允许）；D can be allowed 语义不对（说的是危险，应用 mustn't）。
- **考查知识点**：用法二——mustn't be done 表示"禁止被……"

**第 12 题：答案 B**

- **解析**：These letters 是被寄出的，must be sent。send 的过去分词是 sent。
- **排除法**：A 为主动；C send 不是过去分词；D 缺少 be。
- **考查知识点**：用法二——must be done；不规则动词 send → sent

**第 13 题：答案 C**

- **解析**：The classroom 是被保持清洁的，should be kept clean。keep 的过去分词是 kept。
- **排除法**：A/B 为主动语态；D 缺少 be。
- **考查知识点**：用法三——should be done；keep → kept

**第 14 题：答案 B**

**第 16 题：答案 C**

- **考查知识点**：构成形式——情态被动特殊疑问句；find → found

**第 17 题：答案 B**

- **考查知识点**：用法二——must be done；动词短语被动 turn off

**第 18 题：答案 C**

- **解析**：The environment 是被保护的，should be protected。
- **排除法**：A 为主动；B protect 不是过去分词；D 缺少 be。
- **考查知识点**：用法三——should be done

**第 19 题：答案 B**

- **解析**：the report 是被写的，Must it be written in English? write 的过去分词是 written。
- **排除法**：A 为原形；C wrote 是过去式不是过去分词；D 缺少 be。
- **考查知识点**：构成形式——情态被动疑问句；write → written

**第 20 题：答案 C**

- **解析**：The sick child 是被送去医院的，should be sent。send 的过去分词是 sent。
- **排除法**：A 为主动；B send 不是过去分词；D 缺少 be。
- **考查知识点**：用法三——should be done；send → sent
