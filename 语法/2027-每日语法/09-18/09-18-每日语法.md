# 被动语态——一般将来时的被动（Passive Voice — Simple Future Passive）

> 日期：2027 年 9 月 18 日 | 语法分类：语态 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**一般将来时的被动语态**表示**将来某个时间**将要发生的动作，且主语是**动作的承受者**（不是发出者）。它把“谁做”弱化或省略，突出“事情在将来会怎样被完成 / 被安排”。初中阶段常与 **will**、**be going to** 以及表将来的时间状语连用；有时也与**情态动词 + be + 过去分词**结合，表示“将来应当 / 能够 / 可能被……”。

---

### 2. 构成形式

| 句型 | 结构（structure） | 示例（example） |
|------|------------------|----------------|
| **肯定句（will）** | **will + be + 过去分词 (p.p.)** | The match **will be played** next Sunday. 比赛将于下周日举行。 |
| **肯定句（be going to）** | **am / is / are + going to + be + 过去分词** | A new road **is going to be built** here. 这里将要修一条新路。 |
| **否定句** | **will not (won't) + be + 过去分词** | The plan **won't be changed** easily. 这个计划不会轻易被改变。 |
| **否定句（be going to）** | **am / is / are + not + going to + be + 过去分词** | The meeting **isn't going to be put off**. 会议不打算被推迟。 |
| **一般疑问句** | **Will + 主语 + be + 过去分词 …?** | **Will** the room **be cleaned** tomorrow? 房间明天会被打扫吗？ |
| **一般疑问句（be going to）** | **Am / Is / Are + 主语 + going to be + 过去分词 …?** | **Is** the work **going to be finished** today? 工作今天会被完成吗？ |

**说明**：过去分词规则变化多为词尾 **-ed**；不规则动词需熟记（如 **told, built, held, taken** 等）。

---

### 3. 用法与语境

**用法一：表示将来确定或很可能发生的被动动作（计划、安排）**  
- Our school sports meeting **will be held** next month.  
  我们学校的运动会将于下个月举行。  

**用法二：用 **be going to** 表示“已决定 / 已有迹象”的将来被动**  
- These old books **are going to be given away** to younger students.  
  这些旧书将送给低年级同学。  

**用法三：疑问句询问将来是否会被做**  
- **Will** the lights **be turned off** after class?  
  下课后灯会关吗？  

**用法四：否定句表示将来“不会被……”**  
- Your homework **won't be checked** if you don't hand it in on time.  
  不准时交作业就不会被批改。  

**用法五：含 **by + 执行者**（动作发出者），可省略**  
- The story **will be told** by Lucy at the English corner.  
  这个故事将由露西在英语角讲述。  

**用法六：含情态动词的将来被动（表“应当 / 能够 / 必须”等，常与将来时间呼应）**  
- The baby **should be taken** good care of next week.  
  下周应当好好照顾这个婴儿。  
- The problem **can be solved** before Friday if we work together.  
  若我们一起努力，这个问题在周五前能够被解决。  

---

### 4. 标志词 / 关键信号

| 标志词 / 短语 | 说明 | 示例 |
|--------------|------|------|
| **tomorrow, tonight, soon** | 泛指不远的将来 | The news **will be reported** on TV tonight. |
| **next week / month / year** | “下一个……” | The bridge **will be built** next year. |
| **in + 一段时间** | “在……之后”（从现在算起） | The work **will be finished** in three days. |
| **be going to** | 打算、计划 | The party **is going to be held** on Saturday. |
| **情态动词 + be + p.p.** | 义务、可能、能力等（可配合将来语境） | Such mistakes **must not be made** again. |

---

### 5. 主动变被动（一般将来时）的步骤

1. **找宾语**：把主动句的**宾语**变成被动句的**主语**。  
2. **换谓语**：用 **will be / be going to be + 过去分词**（随新主语的人称变化调整 **am/is/are**）。  
3. **处理原主语**：原主语常放在 **by + 名词 / 代词宾格** 后；若不言自明可省略。  
4. **保留时间、地点等状语**：位置可放在句末或按习惯调整。  

**例**：They will finish the homework tomorrow.  
→ The homework **will be finished** (by them) tomorrow.

---

### 6. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *The meeting **will held** next week.* | The meeting **will be held** next week. | 被动语态中 **will** 后须接 **be + 过去分词**，不能省略 **be**。 |
| 2 | *The house **will be build** soon.* | The house **will be built** soon. | **build** 的过去分词是 **built**，不是原形。 |
| 3 | *The problem **will be talked** in class.* | The problem **will be talked about** in class. | **talk about** 是短语动词，变被动时**介词不能丢**。 |
| 4 | *A letter **will going to be sent**.* | A letter **is going to be sent**. | **will** 与 **be going to** 不宜杂糅；**be going to** 结构中用 **am/is/are**。 |
| 5 | *The work **can finished** tomorrow.* | The work **can be finished** tomorrow. | 情态动词后的被动结构是 **情态动词 + be + 过去分词**。 |

### 逐题解析

**第 1 题：答案 A**

- **解析**：时间状语 **next Saturday** 表将来；主语 **The songs** 是“被唱”，谓语用 **will be + 过去分词**，即 **will be sung**。  
- **排除法**：B、D 是主动语态；C **are sung** 多表习惯或一般现在时的被动，与“下周六”的将来语境不如 A 贴切。  
- **考查知识点**：知识点精讲 → **用法一**（**will be + p.p.** 表将来被动）。

**第 2 题：答案 B**

- **解析**：**homework** 是“被完成”，将来用 **will be finished**。  
- **排除法**：A、C、D 均为主动或进行，不符合“作业被完成”。  
- **考查知识点**：**构成形式** → 肯定句 **will + be + 过去分词**。

**第 3 题：答案 C**

- **解析**：**rules** 与 **explain** 是被动关系，将来一般疑问句结构为 **Will + 主语 + be + 过去分词**，即 **be explained**。  
- **排除法**：A 缺 **-ed**；B **explaineded** 拼写错误；D 不是被动结构。  
- **考查知识点**：**构成形式** → 一般疑问句中的 **be + 过去分词**。

**第 4 题：答案 D**

- **解析**：**park** 是“被建造”，**be going to** 的被动为 **is / are going to be + 过去分词**，主语单数用 **is going to be built**。  
- **排除法**：A、B、C 均为主动“建造”，主语不能执行“建自己”。  
- **考查知识点**：**用法二**（**be going to be + p.p.**）。

**第 5 题：答案 A**

- **解析**：**sports meeting** “被举办”，将来被动 **will be held**。  
- **排除法**：B、D 为主动；C **is held** 常表经常性，不如 **will be held** 贴合 **next month**。  
- **考查知识点**：**标志词** **next month** + **will be + p.p.**。

**第 6 题：答案 B**

- **解析**：**talk about** 为短语动词，被动为 **will be talked about**；题干在空格后已给出 **about**，故空格处填 **will be talked**。  
- **排除法**：A、C 为主动；D **are talking** 为进行时，不符合将来被动。  
- **考查知识点**：**易错点 3**（短语动词被动不可丢介词）。

**第 7 题：答案 C**

- **解析**：“你被老师带去办公室”，承受者是 **you**，用 **will be taken**。  
- **排除法**：A 为主动；B **will took** 结构错误；D **are take** 语法错误。  
- **考查知识点**：**主动变被动**（宾语变主语 + **will be + p.p.**）。

**第 8 题：答案 D**

- **解析**：**lights** 与 **turn off** 是被动，将来疑问句：**Will the lights be turned off …?**  
- **排除法**：A、C 为主动；B **Are; turned** 不是本题最佳的一般将来被动疑问形式。  
- **考查知识点**：**用法三**（疑问句 **Will … be + p.p.**）。

**第 9 题：答案 A**

- **解析**：**story** “被讲述”，**next week** 表将来：**will be told**。  
- **排除法**：B 为一般现在被动，语境偏弱；C、D 不是被动谓语。  
- **考查知识点**：**用法一**与 **用法五**（**by** 可省略）。

**第 10 题：答案 B**

- **解析**：**tickets** “被售完”，**by tomorrow afternoon** 表将来截止：**will be sold**。  
- **排除法**：A、C 为主动；D **sold** 不能单独作将来谓语。  
- **考查知识点**：**构成形式** + 标志词 **by + 时间**（表将来截止）。

**第 11 题：答案 C**

- **解析**：**letter** “被寄送”，**be going to** 被动：**is going to be sent**（主语 **This letter** 单数）。  
- **排除法**：A 为主动；B 缺 **is**；D **is sent** 一般现在被动，与“周五前将要”不符。  
- **考查知识点**：**用法二** + **易错点 4**（**be going to** 结构要完整）。

**第 12 题：答案 D**

- **解析**：情态动词 **can** 后的被动为 **can be finished**。  
- **排除法**：A 为主动；B **be finish** 错误；C **can finished** 缺 **be**。  
- **考查知识点**：**用法六** + **易错点 5**（情态动词 + **be + p.p.**）。

**第 13 题：答案 A**

- **解析**：**trees** “被种植”，将来被动疑问：**Will more trees be planted …?**  
- **排除法**：B、C、D 均不是正确的将来被动疑问结构。  
- **考查知识点**：**构成形式** → **Will + 主语 + be + 过去分词**。

**第 14 题：答案 B**

- **解析**：**classroom** “被打扫”，**next term** 表将来：**will be cleaned**。  
- **排除法**：A、C 为主动；D 过去式不能表达将来安排。  
- **考查知识点**：**用法一**（计划中的将来被动）。

**第 15 题：答案 C**

- **解析**：**project** “被完成”，**next month**：**will be completed**。  
- **排除法**：A、D 为主动；B **will completed** 结构错误。  
- **考查知识点**：**will be + 过去分词**（**complete → completed**）。

**第 16 题：答案 D**

- **解析**：**take good care of** 变被动时，常用 **… be taken good care of**；结合 **next week** 表将来，用 **will be taken**。全句意为：婴儿在下周会得到好好照顾。  
- **排除法**：A **will take** 若选则成“婴儿去照顾”，逻辑不通；B **will be take** 缺过去分词 **taken**；C **took** 为过去式主动，且主语不能执行该动作。  
- **考查知识点**：**用法六**（被动结构）+ **一般将来时** **will be + p.p.**（**take → taken**）。

**第 17 题：答案 A**

- **解析**：**bridge** 与 **build** 为被动关系；**When** 引导的特殊疑问句中，将来被动常用 **will … be built**。  
- **排除法**：B **will build** 为主动；C **is; built** 易混为一般现在被动，与“两年后”的将来不符；D **does; build** 结构错误。  
- **考查知识点**：**标志词** **in two years** + **will be + p.p.**（特殊疑问句语序）。

**第 18 题：答案 B**

- **解析**：**mistakes** 是“不被允许（出现）”，将来否定被动：**won't be allowed**。  
- **排除法**：A、C 为主动，主语是 **mistakes** 不能“允许”；D **aren't allowed** 为一般现在被动，**next time** 更宜用将来形式。  
- **考查知识点**：**用法四**（**won't be + p.p.**）+ **拔高**（否定被动与语境）。

**第 19 题：答案 C**

- **解析**：**news** “被播报”，**this evening** 表将来安排：**will be reported**。  
- **排除法**：A、B 为主动或形式错误；D **is reporting** 为进行时，且 **news** 通常不作“正在报道”的主语。  
- **考查知识点**：**标志词** **this evening** + **will be + p.p.**。

**第 20 题：答案 D**

- **解析**：**sports day** “被推迟”，**put off** 的被动为 **be put off**；主句表将来结果，用 **will be put off**。  
- **排除法**：A **will put off** 为主动，主语不能“推迟自己”；B **puts off** 为主动且时态不对；C **is put off** 为一般现在被动，条件句中主句表将来结果宜用 **will**。  
- **考查知识点**：**用法一** + 短语动词 **put off** 的被动形式（**be put off**）。
