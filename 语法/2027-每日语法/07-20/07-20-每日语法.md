# 定语从句——that 与 which 在限制性定语从句中的区别（that vs which in Restrictive Attributive Clauses）

> 日期：2027 年 7 月 20 日 | 语法分类：从句 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**限制性定语从句**紧跟先行词，**无逗号**隔开，用来**限定**先行词是“哪一个/哪一些”。先行词为**物**时，关系代词常用 **that** 或 **which**，二者在多数情况下**可以互换**；但在一些**固定习惯**和**规则**下**只能用 that** 或**优先用 which**（如介词后）。

### 2. 构成形式

| 情况 | 关系词 | 示例 |
|------|--------|------|
| 一般“物”作先行词 | that / which 常可换 | The book **that/which** I borrowed is interesting. |
| 不定代词等作先行词 | 多用 **that** | Everything **that** he said sounded polite. |
| 先行词被最高级、序数词、the only 等修饰 | 多用 **that** | This is the best film **that** I've seen this year. |
| 介词位于关系代词前 | 用 **which**（不用 that） | This is the room in **which** we held the meeting. |

**注意**：关系代词在从句中作宾语时常可省略（口语/书面均可出现）。

### 3. 用法与语境

- **用法一：普通指物，that 与 which 均可**
  - 例：The bike **that/which** stands by the gate is mine. （门口那辆自行车是我的。）

- **用法二：先行词为 *all, everything, nothing, something, anything, little, much* 等不定代词，关系词常用 that**
  - 例：There is little **that** we can do about the noise tonight. （今晚我们对这噪音几乎没办法。）

- **用法三：先行词被序数词、最高级、*the only, the very, the same* 等修饰，关系词常用 that**
  - 例：He is the only student **that** passed the difficult quiz. （他是唯一通过这次难小测的学生。）

- **用法四：先行词既有人又有物，用 that**
  - 例：They talked about the teachers and topics **that** impressed them most. （他们谈论最让他们印象深刻的老师与话题。）

### 4. 标志词 / 关键信号

| 标志词/短语 | 说明 | 关系词倾向 |
|------------|------|------------|
| the first / second / last… + n. | 序数词修饰先行词 | **that** |
| the best / most important… | 最高级等 | **that** |
| all / everything / nothing… | 不定代词 | **that** |
| 介词 + 关系代词 | 正式语体 | **which** |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *the book which I bought it* | the book **which/that** I bought | 关系代词已作宾语，**不要**再加 *it*。 |
| 2 | *This is all which we need.* | all **that** we need | *all* 作先行词时常用 **that**。 |
| 3 | *the room in that* | the room in **which** / **that…**（介词不放 that 前） | **介词后**不用 *that*，用 *which*。 |

### 逐题解析

**第 1 题：答案 B**

- **解析**：先行词 *dictionary* 为物，限制性定语从句中 *which* 与 *that* 常可互换；本题正确项为 *which*。
- **排除法**：*who* 指人；*what* 不引导定语从句。
- **考查知识点**：用法一；基本概念。

**第 2 题：答案 C**

- **解析**：先行词为不定代词 *much*，关系代词习惯用 **that**。
- **排除法**：易错点 2 同类思路；*which* 在此类先行词上不优先；*what/who* 不符。
- **考查知识点**：用法二；*much that…*。

**第 3 题：答案 D**

- **解析**：先行词被 *the same* 修饰，关系代词常用 **that**。
- **排除法**：*which* 在此结构中不如 *that* 典型；*what/who* 不符。
- **考查知识点**：用法三；标志词 *the same*。

**第 4 题：答案 A**

- **解析**：先行词 *movie* 为物，从句缺宾语，用 **that**（也可用 *which*，选项中给 *that*）。
- **排除法**：*who/whom/whose* 用于人，不修饰 *movie*。
- **考查知识点**：用法一。

**第 5 题：答案 B**

- **解析**：先行词 *Everything* 为不定代词，关系代词用 **that**。
- **排除法**：*which* 不用于 *everything* 的典型考法；*what* 非定语从句关系词。
- **考查知识点**：用法二；标志词 *everything*。

**第 6 题：答案 C**

- **解析**：先行词被序数词 *second* 修饰，关系代词常用 **that**。
- **排除法**：*which* 在此类题中次于 *that*；*what/who* 不符。
- **考查知识点**：用法三；序数词。

**第 7 题：答案 D**

- **解析**：先行词 *shoes* 为物，从句缺主语，用 **that**（或 *which*，选项中为 *that*）。
- **排除法**：*who/whom* 指人；*what* 不用。
- **考查知识点**：用法一。

**第 8 题：答案 A**

- **解析**：先行词 *little money* 含 *little* 表“少而不定”，关系代词常用 **that**。
- **排除法**：*what* 不引导定语从句；*who/whom* 指人。
- **考查知识点**：用法二；*little*。

**第 9 题：答案 B**

- **解析**：先行词 *museum* 为物，从句缺宾语，**which** 与 **that** 皆可；本题正确项为 *which*。
- **排除法**：*who/whom* 指人；*what* 不用。

**第 10 题：答案 C**
