# 一般过去时——used to 与 be used to 的区别（Past Habit — used to vs be used to vs get used to）

> 日期：2027 年 10 月 1 日 | 语法分类：时态 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

中考里常把 **used to**、**be used to**、**get used to** 放在一起考，但它们**意思和结构完全不同**：

- **used to + 动词原形**：谈**过去的习惯或状态**，暗含**现在已改变**（“过去常常 / 过去曾经是”）。
- **be used to + 名词 / 代词 / 动名词 (doing)**：谈**现在的习惯与适应**（“习惯于……”），**to 是介词**。
- **be used to + 动词原形**：多出现在**被动语态**里，表示“**被用来做某事**”（**use** 的被动 + **不定式表用途**），不要与上面“习惯于”混淆。
- **get used to + 名词 / doing**：表示**逐渐变得习惯**，常可与 **become used to** 换用，强调**过程**。

记口诀：**used to do** 看**过去**；**be / get used to doing / sth** 看**习惯**；**…is used to do…** 看**用途（被动）**。

---

### 2. 构成形式

| 结构 | 形式要点 | 示例 |
|------|-----------|------|
| **used to do** | **used** 不变形；否定常考 **didn't use to do** / **used not to do**；疑问用 **Did + 主语 + use to do…?** | He **used to walk** to school.<br>I **didn't use to** like milk.<br>**Did** you **use to** live here? |
| **be used to (doing) / sth** | **be** 随人称时态变化（am / is / are / was / were）；**to 后接名词或 doing** | She **is used to** the noise.<br>We **are used to** **getting** up early. |
| **get / become used to (doing) / sth** | **get** 有时态变化；**to** 同样是介词 | You'll **get used to** the weather.<br>He **got used to** **working** online. |
| **be used to do**（被动表用途） | **主语 + be + used + to + 动词原形** | Wood **is used to make** paper.<br>Knives **are used to cut** things. |

**注意**：没有 *be used to do sth* 表示“习惯于做某事”这种初中常考结构；“习惯于”一定是 **be used to doing / sth**。

---

### 3. 用法与语境

**用法一：used to do —— 过去的习惯 / 过去的状态（现在不再）**  
- I **used to play** the piano, but I stopped last year.  
  我过去常弹钢琴，但去年停了。  
- There **used to be** a bookstore on this street.  
  这条街上过去有一家书店。

**用法二：be used to (doing) / sth —— 已经习惯**  
- My cousin **is used to** **eating** with chopsticks now.  
  我表弟现在已经习惯用筷子吃饭。  
- Are you **used to** the life in the dorm?  
  你习惯宿舍生活了吗？

**用法三：get used to (doing) / sth —— 逐渐习惯**  
- It took me a month **to get used to** the new timetable.  
  我花了一个月才习惯新课表。  
- She **is getting used to** **speaking** English in class.  
  她正逐渐习惯在课堂上说英语。

**用法四：be used to do —— 被用来做（被动 + 不定式）**  
- This room **is used to store** old books.  
  这间屋子被用来存放旧书。  
- Water **can be used to put out** small fires in some cases.  
  在某些情况下水可用来扑灭小火。

**用法五：否定与疑问（used to）**  
- He **didn't use to** wear glasses. = He **used not to** wear glasses.  
  他过去不戴眼镜。  
- **Did** she **use to** swim here every summer?  
  她过去每个夏天都在这里游泳吗？

---

### 4. 标志词 / 关键信号

| 标志词 / 语境 | 说明 | 示例 |
|--------------|------|------|
| **but now / these days / no longer** | 与 **used to** 呼应，表“过去与现在对比” | I **used to** be shy, **but now** I'm outgoing. |
| **at first… later / soon** | 常与 **get used to** 搭配 | **At first** it was strange; **later** I **got used to** it. |
| **already / still / not… yet** | 常与 **be used to** 搭配 | I'm **still not used to** the spicy food here. |
| **主语是工具 / 材料 / 机器** | 优先考虑 **be used to do**（表用途） | This tool **is used to open** the windows. |
| **Did / didn't** | 帮助判断 **used to** 的疑问 / 否定 | **Did** you **use to** walk to school? |

---

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *I am used to **get** up early.* | I **am used to getting** up early. | 表“习惯”时 **to** 是介词，后接 **doing / 名词**。 |
| 2 | *I used to **living** here.* | I **used to live** here. | **used to** 表过去习惯时后接**动词原形**，不是动名词。 |
| 3 | *He is used to **do** homework every day.*（想表达“习惯做作业”） | He **is used to doing** homework every day. | “每天习惯做”用 **be used to doing**；**be used to do** 是“被用来做”。 |
| 4 | *Did you **used to** play football?* | **Did** you **use to** play football? | 疑问句里 **Did** 已体现过去，后面用 **use**（原形），不用 **used**。 |
| 5 | 混淆 **used to do** 与 **be used to doing** | **used to do**：过去常常。<br>**be used to doing**：现在已习惯。 | 看时间：谈“过去且现在变了”用 **used to**；谈“已经适应”用 **be / get used to**。 |

### 逐题解析

**第 1 题：答案 B**

- **解析**：题干有 **when he was young** 与 **but he seldom goes now**，表示**过去的习惯、现在不再**，应选 **used to + 动词原形** **used to go**。
- **排除法**：A、D 把 **be used to** 误接动词原形表“习惯”，结构错误；C **gets used to** 后应接 **going**，且语义是“逐渐习惯”，与“年轻时每周去”不符。
- **考查知识点**：知识点精讲 → **用法一：used to do**；易错点 5（与 be used to 区分）。

**第 2 题：答案 C**

- **解析**：**It took sb. some time to do sth.** 这里需要“花几周去**逐渐适应**校规”，用 **to get used to following**（**get used to + doing**）。
- **排除法**：A **used to follow** 表“过去常常遵守”，与 **took her a few weeks** 的过程不匹配；B、D 结构不成立。
- **考查知识点**：知识点精讲 → **用法三：get used to (doing)**。

**第 3 题：答案 A**

- **解析**：**be not used to + 名词**：**I'm still not used to the heavy traffic**，“我仍不习惯拥堵的交通”。
- **排除法**：B **use to** 不能这样接名词；C **used for** 表用途，不表“习惯”；D **using to** 不是正确搭配。
- **考查知识点**：知识点精讲 → **用法二：be used to sth**；标志词 **still not**。

**第 4 题：答案 D**

- **解析**：刀是工具，表示“**被用来切**”，用被动表用途 **are used to cut**（**be used to + 动词原形**）。
- **排除法**：A **are used to cutting** 是“习惯于切”，主语通常是人；B **used to cut** 会变成“过去常常切”，不合逻辑；C **get used to cut** 结构错误。
- **考查知识点**：知识点精讲 → **用法四：be used to do**；易错点 3。

**第 5 题：答案 B**

- **解析**：**need to** 后接不定式；此处表示“需要**逐渐习惯**早起”，用 **get used to getting up**（**to** 为介词，接 **doing**）。
- **排除法**：A **use to get up** 不是正确结构；C **be used to get up** 把“用途/习惯”结构混在一起；D **used to get up** 表“过去常常起床”，与“给我一点时间”的语境不符。
- **考查知识点**：知识点精讲 → **用法三：get used to doing**。

**第 6 题：答案 A**

- **解析**：**but now he plays** 对比过去与现在，否定过去的状态用 **didn't use to like**。
- **排除法**：B **wasn't used to like** 结构错误；C、D 把 **used to** 与 **be used to doing** 混用。
- **考查知识点**：知识点精讲 → **用法五：否定 didn't use to do**；标志词 **but now**。

**第 7 题：答案 C**

- **解析**：在国外三个月后，表示**那段时间里已经习惯**独居，用 **was used to living**（**be used to + doing**）。
- **排除法**：A **used to live** 强调“过去常常住”，不强调“适应”；B **was used to live** 易被误当成被动表用途，语义不通；D **got used to live** 应为 **living**。
- **考查知识点**：知识点精讲 → **用法二：be used to doing**。

**第 8 题：答案 D**

- **解析**：手机**被用来做**很多事，用 **are used to do**（被动 + 不定式表用途）。
- **排除法**：A、B 主语与主动形式不匹配；C **are used to doing** 表示“习惯于做”，主语通常是人。
- **考查知识点**：知识点精讲 → **用法四：be used to do**。

**第 9 题：答案 A**

- **解析**：**used to** 的疑问句：**Did + 主语 + use to + 动词原形** → **Did you use to play**。
- **排除法**：B **Were you used to** 后应接 **playing** 等，且语义为“你过去习惯吗”，与题干不符；C **Did you used to** 违反易错点 4；D **Were you use to** 结构错误。
- **考查知识点**：知识点精讲 → **用法五：疑问句**；易错点 4。

**第 10 题：答案 B**

- **解析**：在英国长大，**现在不习惯**用筷子吃饭，用 **isn't used to eating**。
- **排除法**：A **used to eat** 表过去吃饭方式，与原因句逻辑不顺；C **is used to eat** 是被动“被用来吃”；D **gets used to eat** 结构错误，应为 **eating**。
- **考查知识点**：知识点精讲 → **用法二：be used to doing**；易错点 1、3。

**第 11 题：答案 C**

- **解析**：**soon** 提示从“不适应”到“适应”的过程，用一般过去时 **got used to** + **the noisy neighbors**。
- **排除法**：A **used to** 后缺动词原形且语义不对；B **was used to** 虽可表“已经习惯”，但 **the first week was hard, but… soon** 更突出变化过程，**got used to** 更贴切；D **am used to do** 结构错误。
- **考查知识点**：知识点精讲 → **用法三：get used to sth**；标志词 **at first / soon**。

**第 12 题：答案 D**

- **解析**：App **被用来帮助**学生复习单词，**is used to help**（表用途的被动）。
- **排除法**：A **used to help** 表“过去常常帮助”；B **is used to helping** 表“习惯于帮助”，语义不对；C **gets used to help** 结构错误。
- **考查知识点**：知识点精讲 → **用法四：be used to do**。

**第 13 题：答案 B**

- **解析**：搬家前**过去住在**小公寓，用 **used to live in**。
- **排除法**：A、D **were used to…** 表“习惯于住”，与“搬家前住过”事实陈述不符；C **got used to live** 结构错误。
- **考查知识点**：知识点精讲 → **用法一：used to do**。

**第 14 题：答案 A**

- **解析**：**now; it feels normal** 表示**现在已经习惯**，用 **am used to**，后接 **the long bus ride**（名词短语，**to** 为介词）。
- **排除法**：B **used to** 表过去，与 **now** 矛盾；C **get used to** 若用一般现在时常表“将要/正在习惯”，此处缺进行或将来形式且不如 **am used to** 贴切；D **was used to do** 为被动用途。
- **考查知识点**：知识点精讲 → **用法二：be used to sth**；易错点 5。

**第 15 题：答案 C**

- **解析**：**He is used to studying**，“他习惯了在严格班级里学习”，空白处填 **used to studying**（与前面的 **is** 构成 **is used to studying**）。
- **排除法**：A **use to studying** 错误；B **used to study** 是“过去常常学习”，与 **is** 不搭配；D **getting used to study** 应为 **studying**。
- **考查知识点**：知识点精讲 → **用法二：be used to doing**；易错点 1。

**第 16 题：答案 D**

- **解析**：木材**被用来造纸**，过去用 **was used to make**。
- **排除法**：A **used to make** 会变成“过去常常制造纸”，主语逻辑不当；B **was used to making** 是“习惯于制造”；C **got used to make** 结构错误。
- **考查知识点**：知识点精讲 → **用法四：be used to do**（过去时）。

**第 17 题：答案 A**

- **解析**：**There used to be** 表“过去曾有”，与 **closed last year** 呼应。
- **排除法**：B、C 结构混乱；D **used to being** 不能用于 **there** 存在句的规范表达。
- **考查知识点**：知识点精讲 → **用法一：used to do**（**there used to be**）。

**第 18 题：答案 B**

- **解析**：**hasn't… yet** 表示“还没习惯时差”，用现在完成时 **hasn't got used to**（**got** 为 **get** 的过去分词）。
- **排除法**：A **used to** 不能与 **hasn't** 这样搭配表“尚未习惯”；C **been used to do** 为被动用途，语义错；D **use to** 错误。
- **考查知识点**：知识点精讲 → **用法三：get used to sth**；标志词 **not… yet**。

**第 19 题：答案 C**

- **解析**：夜班工人**已经习惯夜班**，用 **is used to night shifts**（**be used to + 名词**）。
- **排除法**：A **used to** 后需动词原形；B **uses to** 结构错误；D **gets used to** 可表逐渐习惯，但此处陈述常态，**is used to** 更自然。
- **考查知识点**：知识点精讲 → **用法二：be used to sth**。

**第 20 题：答案 D**

- **解析**：**but now he takes the bus** 对比过去与现在，过去常常步行上学：**used to walk**。
- **排除法**：A **is used to** 后不能接动词原形；若表“习惯走路”须用 **is used to walking**。B **gets used to walk** 结构错误，应为 **getting used to walking**。C **was used to** 表“过去习惯”时须接 **walking**，不能直接接 **walk**。
- **考查知识点**：知识点精讲 → **用法一：used to do**；易错点 5。
