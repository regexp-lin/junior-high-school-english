# 被动语态——现在完成时的被动（Passive Voice — Present Perfect Passive）

> 日期：2027 年 10 月 4 日 | 语法分类：语态 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**现在完成时的被动语态（Present Perfect Passive）** 表示**到现在为止**，某个动作**已经被完成**，或**经历**被施加于主语；强调**结果、状态或与现在的联系**，而不是单纯说明“过去某一刻谁做了这件事”。主语通常是**动作的承受者**。

---

### 2. 构成形式

| 句型 | 结构 | 示例 |
|------|------|------|
| **肯定句** | **have / has + been + 过去分词（pp）** | The work **has been finished**. |
| **否定句** | **have / has + not + been + pp**（常用缩写 **haven't / hasn't**） | The letter **hasn't been sent** yet. |
| **一般疑问句** | **Have / Has + 主语 + been + pp?** | **Have** you **been invited** to the party? |
| **特殊疑问句** | 疑问词 + **have / has + 主语 + been + pp** | **How many books have** **been sold**? |

**人称提示**：第三人称单数用 **has**，其余用 **have**。**been** 是固定形式，不随人称变化；变化体现在 **have/has** 上。

---

### 3. 用法与语境

**用法一：过去开始的动作到现在为止已经完成或产生影响**  
- The bridge **has been built** in the past five years.  
  这座桥在过去五年里已经建成。（强调“现在已经有了桥”这一结果）  
- So far, all the homework **has been checked** by Miss Li.  
  到目前为止，所有作业都已被李老师批改。

**用法二：经历或变化与“现在”相关（不必指出具体过去时间）**  
- Our classroom **has been painted** recently. It looks bright now.  
  我们教室最近被粉刷过，现在看起来很亮。  
- He **has been told** the news three times, but he still forgets.  
  这消息已经告诉他三遍了，他还是忘。（强调反复经历到现在）

**用法三：与一般过去时被动的对比**  
- **一般过去时被动**（was / were + pp）：常和 **yesterday, last week, in 2020（单纯过去时间点）** 等连用，侧重**过去某时**发生的事。  
  The rules **were explained** to us **last Monday**.  
- **现在完成时被动**（have / has been + pp）：常和 **already, yet, just, so far, recently, in the past few years, since…** 等连用，侧重**与现在的关系**或**从过去延续到现在的一段时期**。  
  The rules **have been explained** to all classes **so far** this term.

---

### 4. 标志词 / 关键信号

| 标志词 / 短语 | 说明 | 示例 |
|--------------|------|------|
| **already** | 已经（常表“比预期早完成”） | The tickets **have already been sold** out. |
| **yet** | 还（多用于否定/疑问） | **Has** the work **been finished** **yet**? |
| **just** | 刚刚 | The window **has just been broken**. |
| **recently / lately** | 最近 | Many trees **have been planted** recently. |
| **so far / up to now / until now** | 到目前为止 | **So far**, no mistakes **have been found**. |
| **in the past / last few years** | 过去几年（到现在为止） | New roads **have been built** in the past few years. |
| **since + 时间点** | 自……以来 | **Since** 2020, the park **has been opened** every day. |
| **for + 一段时间** | 持续多久（与现在仍相关） | The bridge **has been closed** **for** two weeks. |

---

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *The book **has finished**.* | The book **has been finished**. | 书是“被完成”，要用 **been + 过去分词**，不能省略 **been** 或误用主动结构。 |
| 2 | *The rooms **have cleaned**.* | The rooms **have been cleaned**. | 被动语态必须有 **be** 的适当形式；现在完成时被动用 **have/has been + pp**。 |
| 3 | ***Have** the work **been** done?* | **Has** the work **been** done? | **work** 不可数，视作单数，疑问句用 **Has**。 |
| 4 | 有 **yesterday / last year** 却用现在完成时被动 | **Last year** the hall **was** repaired. | **明确的过去时间点**多用**一般过去时被动**；现在完成时被动强调**到现在为止**或与**already/yet/so far** 等搭配。 |
| 5 | *He **has been** invite**d** to…* 拼写混乱 | He **has been invited** to the party. | 过去分词要正确：**invite → invited**，整体结构 **has been + pp** 不能拆错。 |

### 逐题解析

**第 1 题：答案 C**

- **解析**：**in the past few years**（在过去几年里）表示从过去延续到现在的一段时间，强调“到现在为止已经建成”，应使用**现在完成时的被动**：**has been built**（主语 line 单数，用 has）。
- **排除法**：A、B 无法体现“近几年到现在为止”这一时段与现在的联系；D 表示将来，与语境不符。
- **考查知识点**：知识点精讲 → **标志词**（in the past few years）+ **肯定句结构**（has been + pp）。

**第 2 题：答案 A**

- **解析**：**yet** 常用于否定句和疑问句，表示“还（未）”，与**现在完成时**搭配自然：**hasn't been finished yet**（尚未被批改完）。homework 不可数，用 **hasn't**。
- **排除法**：B 中 **already** 多用于肯定句；C 时态与 **yet** 搭配不当；D 过去完成时在此无“过去的过去”语境。
- **考查知识点**：知识点精讲 → **否定句**（has not been + pp）+ **标志词 yet**。

**第 3 题：答案 B**

- **解析**：主语 **the classroom windows** 是复数，助动词用 **Have**；窗户是“被打扫”，用被动 **been cleaned**。**already** 提示动作已完成且与现在相关。
- **排除法**：A 缺 **been**，不是被动；C 结构错误；D **Were cleaned** 可表过去，但与 **already** 强调“到现在为止已经干净”相比，B 更贴切。
- **考查知识点**：知识点精讲 → **一般疑问句**（Have + 主语 + been + pp?）+ **标志词 already**。

**第 4 题：答案 D**

- **解析**：**since we started…** 表示从过去某时起直到现在，主语 **water** 与 **save** 是被动关系，用 **has been saved**（water 不可数，用 has）。
- **排除法**：A、B 不能体现 **since** 带来的“从过去到现在”；C 为主动且结构错误。
- **考查知识点**：知识点精讲 → **标志词 since** + **肯定句**（has been + pp）。

**第 5 题：答案 A**

- **解析**：**just**（刚刚）常与**现在完成时**连用；规则是“被讲解”，用被动 **have been explained**（主语 **rules** 复数，用 **have**）。
- **排除法**：B **were explained** 偏重过去某一时刻，与 **just** 的“刚发生且与现在相关”不如 A 自然；C、D 结构错误。
- **考查知识点**：知识点精讲 → **标志词 just** + **have been + pp**。

**第 6 题：答案 C**

- **解析**：**By now**（到现在为止）是现在完成时的典型信号；**every exercise** 视为单数概念，用 **has been checked**。
- **排除法**：A 一般现在时不能表达“到现在为止已经完成”；B **were** 与单数主语不一致；D **have** 与 **every exercise** 主谓不一致。
- **考查知识点**：知识点精讲 → **标志词 By now** + **has/have been + pp** 与主谓一致。

**第 7 题：答案 B**

- **解析**：**so far this month**（本月到目前为止）用**现在完成时被动**：**has been shown**；**this month** 内多次放映，主语 **film** 单数用 **has**。
- **排除法**：A、C 时态与 **so far** 搭配不当；D 无被动且时态错误。
- **考查知识点**：知识点精讲 → **标志词 so far** + **has been + pp**。

**第 8 题：答案 D**

- **解析**：“今天不能骑车”是**现在的结果**，原因往往是“车已经被拿走/处理了”，用 **has been taken away** 体现对现在的影响。
- **排除法**：A、B、C 均不能正确表达被动或时态与语境。
- **考查知识点**：知识点精讲 → **用法一**（过去动作对现在的影响）+ **has been + pp**。

**第 9 题：答案 A**

- **解析**：**yet** 用于疑问句，询问“是否已经……”；**email** 单数，用 **Has… been sent**。
- **排除法**：B 一般过去时不与 **yet** 典型搭配；C 为主动结构且不符合“邮件被发送”；D 现在进行时被动结构错误。
- **考查知识点**：知识点精讲 → **一般疑问句**（Has + 主语 + been + pp?）+ **yet**。

**第 10 题：答案 C**

- **解析**：**In recent years**（近年来）表示到现在为止的一段时期，用 **have been created**（主语 **corners** 复数）。
- **排除法**：A 无被动；B 偏重单纯过去，不如 C 贴合“近年来到现在为止”；D 过去完成时缺少“过去的过去”语境。
- **考查知识点**：知识点精讲 → **标志词 recently / in recent years** + **have been + pp**。

**第 11 题：答案 B**

- **解析**：答句强调“她现在兴奋”的**原因**是“已经被邀请去发言”，邀请这一动作的结果延续到现在，用 **has been invited**。
- **排除法**：A **was invited** 只强调过去某时，不突出“现在因此兴奋”；C 一般现在时被动不表示“刚刚得知的结果”；D 为主动。
- **考查知识点**：知识点精讲 → **用法二**（经历与现在相关）+ **易错点 4** 与一般过去时的语气差别。

**第 12 题：答案 A**

- **解析**：**Up to now** 与 **have been planted** 搭配；**trees** 复数，用 **have**。
- **排除法**：B 不能体现“到现在为止累计”；C 无被动；D 一般现在时不能表达“已种植”的完成意义。
- **考查知识点**：知识点精讲 → **标志词 up to now** + **have been + pp**。

**第 13 题：答案 D**

- **解析**：现在完成时被动结构为 **has been + 过去分词**，**paint** 的过去分词为 **painted**，只有 D 正确。
- **排除法**：A 缺 **been**；B **been painting** 为进行被动，语义不对；C **paint** 不是过去分词形式。
- **考查知识点**：知识点精讲 → **构成形式** + **易错点 1、2**。

**第 14 题：答案 C**

- **解析**：到说话时“还没被告知”导致走错，强调**到现在为止仍未被告知**，用 **haven't been told**。
- **排除法**：A、D 时态与“因此早到”的因果链不贴切；B **weren't told** 可表示过去某时未被告知，但题干更强调“至今不知道”这一持续状态，C 更符合现在完成时的用法。
- **考查知识点**：知识点精讲 → **否定句**（haven't been + pp）+ **用法与语境**。

**第 15 题：答案 B**

- **解析**：**last Friday afternoon** 是**明确的过去时间**，说明动作发生在过去某一时刻，用**一般过去时被动** **were explained**，不用现在完成时。
- **排除法**：A 现在完成时与 **last Friday** 这类具体过去时间冲突；C、D 结构或语态错误。
- **考查知识点**：知识点精讲 → **用法三**（与一般过去时被动的区别）+ **易错点 4**。

**第 16 题：答案 A**

- **解析**：**Recently** 提示与现在相关，实验室“被配备”电脑，用 **has been equipped**（**lab** 单数）。
- **排除法**：B 可表过去某次，但不如 A 贴合“最近以来现在的状态”；C、D 不符合语境或语态。
- **考查知识点**：知识点精讲 → **标志词 recently** + **has been + pp**。

**第 17 题：答案 D**

- **解析**：**since 2021** 须用**现在完成时**；**shows** 与 **hold** 为被动关系，用 **Have many charity shows been held**。
- **排除法**：A 主动且时态不对；B、C 不能正确搭配 **since 2021** 表达“从过去到现在”。
- **考查知识点**：知识点精讲 → **since** + **一般疑问句**（Have + 主语 + been + pp?）。

**第 18 题：答案 B**

- **解析**：**last month** 为具体过去时间，拆除发生在那一时段，用 **was pulled**（一般过去时被动）。
- **排除法**：A 现在完成时与 **last month** 不宜连用；C、D 语态或时态错误。
- **考查知识点**：知识点精讲 → **用法三**（与一般过去时被动的区别）+ **易错点 4**。

**第 19 题：答案 C**

- **解析**：**So far** 要求现在完成时；**none of the problems** 作主语时，中考常考**谓语单数**，用 **has been solved**。
- **排除法**：A、D 主动或时态不对；B 一般现在时不能表达“到目前为止尚未被解决”。
- **考查知识点**：知识点精讲 → **标志词 so far** + **has been + pp**（主谓一致）。

**第 20 题：答案 D**

- **解析**：**for many years** 表示持续至今的一段时间，且英语是“被使用”，用 **has been spoken**，强调多年来一直延续到现在的状态。
- **排除法**：A 主动错误；B 一般现在时可表事实，但不如 D 贴合 **for many years** 的时段感；C 过去时与 **for many years** 延续到现在矛盾。
- **考查知识点**：知识点精讲 → **标志词 for + 一段时间** + **has been + pp**。
