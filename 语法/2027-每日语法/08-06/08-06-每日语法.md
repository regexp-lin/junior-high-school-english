# 感叹句——What 与 How 引导的感叹句（Exclamatory Sentences — What vs How）

> 日期：2027 年 8 月 6 日 | 语法分类：句型 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

感叹句（exclamatory sentence）用来表达强烈的感情，比如惊喜、赞美、遗憾或兴奋。在英语里，常用 **What** 和 **How** 来引导感叹句，句末一般用感叹号 **!**。简单说：**What** 往往强调“名词（短语）”，**How** 往往强调“形容词或副词”。

### 2. 构成形式

What + (a/an) + 形容词 + 名词 + 主语 + 谓语!  
How + 形容词/副词 + 主语 + 谓语!

| 句型 | 结构 | 示例 |
|------|------|------|
| What + a/an + 形容词 + 可数名词单数 + 主语 + 谓语 | What + a/an + adj. + 可数单数名词 + S + V | What a lovely day it is!（天气多好啊！） |
| What + 形容词 + 可数名词复数 + 主语 + 谓语 | What + adj. + 可数复数名词 + S + V | What beautiful flowers they are!（这些花多美啊！） |
| What + 形容词 + 不可数名词 + 主语 + 谓语 | What + adj. + 不可数名词 + S + V | What fine weather we are having!（这天气多好啊！） |
| How + 形容词 + 主语 + 谓语 | How + adj. + S + V | How kind you are!（你真善良啊！） |
| How + 副词 + 主语 + 谓语 | How + adv. + S + V | How fast he runs!（他跑得真快啊！） |

### 3. 用法与语境

**What + a/an + 形容词 + 可数名词单数**

- 可数名词单数前通常要加 **a** 或 **an**（由后面单词的发音决定）。  
- What a nice gift it is! 这是一件多好的礼物啊！  
- What an honest boy Tom is! 汤姆是个多诚实的男孩啊！

**What + 形容词 + 可数名词复数**

- 名词是复数时，**不加** a/an。  
- What interesting books these are! 这些书多有趣啊！  
- What helpful classmates we have! 我们有多热心的同学啊！

**What + 形容词 + 不可数名词**

- 不可数名词前**不加** a/an。  
- What delicious soup this is! 这汤多鲜美啊！  
- What useful advice you gave me! 你给我的建议多有帮助啊！

**How + 形容词/副词 + 主语 + 谓语**

- **How** 后面直接跟形容词或副词，**不**再跟名词短语。  
- How cold it is today! 今天多冷啊！  
- How carefully she listens! 她听得多认真啊！

**What 与 How 感叹句的相互转换**

- 核心是把“名词短语”和“形容词/副词”的强调点换一下，主谓通常保留。  
- What a tall building it is! → How tall the building is!（这座楼多高啊！）  
- How wonderful the music sounds! → What wonderful music it is!（这音乐多美妙啊！）

### 4. 标志词 / 关键信号

| 判断思路 | 用 What 还是 How | 说明 |
|----------|------------------|------|
| 强调点后紧跟 **名词（短语）** | 多用 **What** | 结构多为 What + (a/an) + 形容词 + 名词 + 主语 + 谓语 |
| 强调点后紧跟 **形容词或副词**，且后面没有名词短语 | 多用 **How** | 结构多为 How + 形容词/副词 + 主语 + 谓语 |
| **可数单数**名词 | What + **a/an** | a/an 由后面单词发音决定，如 an honest boy |
| **复数**或**不可数**名词 | What + **零冠词** | 不用 a/an |
| 句中已有 **the + 名词** 等完整名词短语要感叹 | 常改用 **How + 形容词** | 如 How beautiful the flowers are! |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-------------|-------------|----------|
| 1 | How a good student he is! | What a good student he is! | “a + 可数单数名词”要用 **What** 引导，不能用 How + a |
| 2 | What nice weather it is a! | What nice weather it is! | **weather** 不可数，不能加 **a** |
| 3 | What beautiful the flowers are! | How beautiful the flowers are! | **the flowers** 前已有定冠词，强调形容词用 **How** |
| 4 | How fast time it flies! | How fast time flies! | **How + 副词** 后接正常语序的主谓，不要多写无需要的 **it**（此处 time 为主语） |
| 5 | What interesting story it is! | What an interesting story it is! | **story** 可数单数，形容词 interesting 以元音音素开头，用 **an** |

**第 1 题：答案 A**

- **解析**：**sight**（景象）是可数名词单数，前面有形容词 **beautiful**，结构为“What + a/an + 形容词 + 可数单数 + 主语 + 谓语”，**beautiful** 以辅音音素开头，用 **What a**。
- **排除法**：B、D 的 **How** 后不能直接加 **a** 再加名词短语；C **What** 后缺 **a**，与可数单数 **sight** 不搭配。
- **考查知识点**：知识点精讲 **2. 构成形式** 表中第一行；“**What + a/an + 形容词 + 可数名词单数**”。

**第 2 题：答案 B**

- **解析**：被强调的是副词 **carefully**（认真地），结构为“**How + 副词 + 主语 + 谓语**”，用 **How**。
- **排除法**：**What** 类引导词后需接名词短语，此处紧跟副词，故 A、C、D 均不适用。
- **考查知识点**：知识点精讲 **2. 构成形式** 表中第五行；“**How + 副词 + 主语 + 谓语**”。

**第 3 题：答案 C**

- **解析**：**news**（消息）不可数，结构为“What + 形容词 + 不可数名词 + 主语 + 谓语”，不加 **a**，用 **What**。
- **排除法**：A **How** 后应接形容词/副词，不能直接接 **exciting news** 这种名词短语；B **How a** 结构错误；D **What a** 不能用于不可数名词 **news**。
- **考查知识点**：知识点精讲 **3. 用法与语境** 中“**What + 形容词 + 不可数名词**”。

**第 4 题：答案 D**

- **解析**：**boy** 是可数单数，**honest** 以元音音素开头，用 **an**，即 **What an honest boy he is!**。
- **排除法**：A **What** 缺少 **a/an**；B **What a** 用于辅音音素开头的形容词前，与 **honest** 不符；C **How** 后不能接 **a/an + 名词**。
- **考查知识点**：知识点精讲 **3. 用法与语境** 中“**What + a/an + 形容词 + 可数名词单数**”；**5. 易错点** 第 5 条（**an** 的选用）。

**第 5 题：答案 A**

- **排除法**：B **What** 后需接名词短语，此处 **hard** 为形容词作状语含义；C、D 带 **a**，与形容词 **hard** 不搭配。
- **考查知识点**：知识点精讲 **3. 用法与语境** 中“**How + 形容词/副词 + 主语 + 谓语**”。

**第 6 题：答案 B**

- **排除法**：A **How** 后不能直接接 **delicious apples**；C **How a**、D **What a** 均不能与复数名词 **apples** 搭配。

**第 7 题：答案 C**
