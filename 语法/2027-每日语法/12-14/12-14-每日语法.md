# 状语从句——结果状语从句 so...that / such...that（Adverbial Clauses of Result — so...that / such...that）

> 日期：2027 年 12 月 14 日 | 语法分类：从句 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

结果状语从句表示主句动作产生的**结果**，由 **so...that** 和 **such...that** 引导，意为"如此……以至于……"。二者核心区别在于：**so 修饰形容词/副词**，**such 修饰名词（短语）**。这是中考高频考点，尤其常考 so 与 such 的互换。

### 2. 构成形式

| 结构 | 修饰对象 | 句型 | 示例 |
|------|---------|------|------|
| so...that | 形容词/副词 | so + adj./adv. + that... | He is so tall that he can reach the shelf. |
| such...that | 名词（短语） | such + a/an + adj. + 名词 + that... | He is such a tall boy that he can reach the shelf. |
| such...that | 不可数名词/复数名词 | such + adj. + 名词(复/不可数) + that... | It was such bad weather that we stayed home. |

| so 的特殊结构 | 示例 |
|--------------|------|
| so + adj. + a/an + 名词 + that | He is so tall a boy that... |
| so + many/much/few/little + 名词 + that | There are so many people that... |

### 3. 用法与语境

- **用法一**：so + 形容词/副词 + that
  - 例：She spoke so fast that I couldn't understand her.（她说得太快了以至于我听不懂。）
  - 例：He is so clever that he can solve the problem easily.（他如此聪明以至于能轻松解决问题。）

- **用法二**：such + a/an + 形容词 + 可数名词单数 + that
  - 例：It was such a cold day that we decided to stay at home.（天太冷了以至于我们决定待在家里。）
  - 例：She is such a beautiful girl that everyone likes her.（她是如此漂亮的女孩以至于大家都喜欢她。）

- **用法三**：such + 形容词 + 不可数名词/可数名词复数 + that
  - 例：It was such bad weather that we couldn't go out.（天气太糟了以至于我们不能出去。）
  - 例：They are such good students that the teacher is proud of them.（他们是如此优秀的学生以至于老师为他们骄傲。）

- **用法四**：so 与 such 的互换
  - so + adj. + a/an + 名词 = such + a/an + adj. + 名词
  - 例：He is so good a student that... = He is such a good student that...

- **用法五**：so many/much/few/little 后用 so 而非 such
  - so many + 可数名词复数：There were so many people that...
  - so much + 不可数名词：He has so much money that...
  - so few + 可数名词复数：There are so few books that...
  - so little + 不可数名词：He has so little time that...

### 4. 标志词 / 关键信号

| 判断依据 | 选择 | 示例 |
|----------|------|------|
| 后接形容词/副词 | so | so beautiful / so quickly |
| 后接 a/an + adj. + 名词 | such | such a good boy |
| 后接 adj. + 复数名词/不可数名词 | such | such interesting books |
| many/much/few/little + 名词 | so | so many people |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | It was such cold that we stayed home. | It was so cold that we stayed home. | cold 是形容词，后无名词，用 so |
| 2 | He is so a good student that... | He is such a good student that... 或 He is so good a student that... | so 修饰形容词时 a 放在 adj. 后面 |
| 3 | There were such many people that... | There were so many people that... | many/much/few/little 前用 so |
| 4 | She ran such fast that... | She ran so fast that... | fast 是副词，用 so |
| 5 | It was so a bad weather that... | It was such bad weather that... | weather 不可数无 a/an，用 such |

**第 1 题：答案 A**

- **解析**：tired 是形容词，用 so...that"如此累以至于走不动了"。
- **排除法**：B. such 后接名词；C. too...to 结构不同；D. very 不与 that 搭配。
- **考查知识点**：知识点精讲 → 用法一。

**第 2 题：答案 B**

- **解析**：such a cold day，such + a + adj. + 名词 + that。

- **考查知识点**：知识点精讲 → 用法二。
