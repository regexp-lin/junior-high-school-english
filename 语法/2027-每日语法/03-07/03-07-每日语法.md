# 目的状语从句——so that / in order that（Purpose Clauses）

> 日期：2027 年 03 月 07 日 | 语法分类：从句 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**目的状语从句（purpose clause）** 表示主句动作的**目的**，即"为了……"。常见引导词有 **so that** 和 **in order that**。从句中通常含有情态动词 **can/could/may/might/will/would**。

---

### 2. so that 引导目的状语从句

#### 2.1 基本结构

> **主句 + so that + 主语 + can/could/may/might/will/would + 动词原形**

| 示例 |
|------|
| He got up early **so that** he **could** catch the bus. |
| I'll write it down **so that** I **won't** forget it. |
| She studies hard **so that** she **can** pass the exam. |

#### 2.2 注意事项

- so that 从句中通常含有情态动词（can/could/may/might/will/would）。
- so that 引导的目的状语从句一般放在**主句之后**。
- so that 还可以引导**结果状语从句**（表示"以至于/结果"），需根据语境区分。

---

### 3. in order that 引导目的状语从句

#### 3.1 基本结构

> **In order that + 主语 + can/could/may/might..., 主句**
> 或 **主句 + in order that + 主语 + can/could/may/might...**

| 示例 |
|------|
| **In order that** we **could** arrive on time, we left early. |
| He works hard **in order that** he **may** support his family. |

#### 3.2 特点

- in order that 比 so that 更正式。
- in order that 可以放在句首或句末。
- 从句中也需含情态动词。

---

### 4. so that（目的）vs. so that（结果）

| 类型 | 含义 | 从句特征 | 示例 |
|------|------|---------|------|
| **目的** | 为了…… | 从句含 can/could/may/might 等 | He spoke loudly **so that** everyone **could** hear him. |
| **结果** | 以至于/结果 | 从句用陈述句，无情态动词 | He spoke so loudly **that** everyone heard him. |

**对比：**
- 目的：He studied hard **so that** he **could** pass the exam. （为了通过考试）
- 结果：He studied **so** hard **that** he passed the exam. （学习如此努力以至于通过了）

---

### 5. 简化为不定式结构

目的状语从句可以简化为 **to do / in order to do / so as to do**：

| 从句形式 | 简化形式 | 示例 |
|---------|---------|------|
| so that + 主语 + can... | **to do / in order to do** | He got up early **to/in order to** catch the bus. |
| so that + 主语 + can't... | **so as not to / in order not to** | He spoke softly **so as not to / in order not to** wake the baby. |

#### 简化条件

- 主句和从句的主语**相同**时才能简化。
  - ✅ He works hard so that **he** can pass. → He works hard **to** pass.
  - ❌ He works hard so that **his family** can live well. （主语不同，不能简化）

#### to do / in order to do / so as to do 的区别

| 结构 | 位置 | 否定形式 |
|------|------|---------|
| **to do** | 句首或句末 | 不常用 not to do |
| **in order to do** | 句首或句末 | in order **not** to do |
| **so as to do** | 只能放句末 | so as **not** to do |

---

### 6. 其他表目的的结构（拓展）

| 结构 | 示例 |
|------|------|
| for + 名词/动名词 | I went there **for** a meeting. |
| for the purpose of + 动名词 | He went abroad **for the purpose of** studying. |
| with the aim of + 动名词 | She practices **with the aim of** improving her skills. |

---

### 7. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-------------|-------------|----------|
| 1 | He studied hard so that pass the exam. | He studied hard so that he **could** pass the exam. | so that 从句需有主语和情态动词 |
| 2 | So as to catch the bus, he got up early. | **In order to** catch the bus, he got up early. | so as to 不能放句首 |
| 3 | He spoke softly in order to not wake the baby. | He spoke softly **in order not to** wake the baby. | not 放在 to 前面 |
| 4 | He works hard so that his family to live well. | He works hard so that his family **can** live well. | so that 从句用完整句子 |
| 5 | I study hard so as to I can pass. | I study hard **so that** I can pass. / I study hard **so as to** pass. | so as to 后直接接动词原形 |

### 逐题解析

**第 1 题：答案 A**

- **解析**："他早起为了赶上第一班公交"，目的关系用 **so that** + could。
- **排除法**：B 原因；C 让步；D 条件。
- **考查知识点**：so that 引导目的状语从句。

**第 2 题：答案 B**

- **解析**：后面直接接动词原形 pass，用 **in order to**。
- **排除法**：A so that 后需要完整从句；C 原因；D 让步。
- **考查知识点**：in order to + 动词原形。

**第 3 题：答案 C**

- **解析**："我们大声说话为了每个人都能听到"，目的用 **so that**。
- **排除法**：A 原因；B 让步；D 条件。
- **考查知识点**：so that + 情态动词。

**第 4 题：答案 B**

- **解析**：放在句首，后接动词原形。**In order to** 可放句首，so as to **不能**放句首。
- **排除法**：A so that 后需从句；C so as to 不能放句首（易错点 2）；D 因 C 错所以不对。
- **考查知识点**：in order to 可放句首，so as to 不可。

**第 5 题：答案 A**

- **解析**："写下来为了不会忘记"，目的关系用 **so that**。
- **排除法**：B 原因；C 条件；D 让步。
- **考查知识点**：so that 引导目的从句。

**第 6 题：答案 C**

- **解析**：后面直接接动词原形 catch，用 **in order to**。
- **排除法**：A/B 后面需要完整从句；D 不存在 so as that。
- **考查知识点**：in order to + 动词原形。

**第 7 题：答案 B**

- **解析**："她关窗为了雨水不会进来"，目的用 **so that**。
- **排除法**：A 原因；C 让步；D 条件。
- **考查知识点**：so that + would。

**第 8 题：答案 A**

- **解析**：In order not to be late — **In order** 放句首表目的。
- **排除法**：B so as 不能放句首；C so that 后需完整从句；D 原因。
- **考查知识点**：in order (not) to 可放句首。

**第 9 题：答案 A**

- **解析**："他存钱为了能买新电脑"，目的用 **so that** + could。
- **排除法**：B 让步；C 原因；D 条件。
- **考查知识点**：so that 引导目的从句。

**第 10 题：答案 B**

- **解析**：后接动词原形 make，用 **in order to**。
- **排除法**：A 后需完整从句；C 原因；D 让步。
- **考查知识点**：in order to + 动词原形。

**第 11 题：答案 A**

- **解析**："他说得慢为了学生们能理解"，目的用 **so that**。注意 so...that 是"如此……以至于"（结果），结构不同。
- **排除法**：B so...that 用于 so + adj./adv. + that，结构不合；C 原因；D 让步。
- **考查知识点**：so that（目的）vs. so...that（结果）。

**第 12 题：答案 C**

- **解析**："她躲在门后为了没人发现她"，目的用 **so that**。
- **排除法**：A 原因；B 让步；D 条件。
- **考查知识点**：so that + would。

**第 13 题：答案 B**

- **解析**：后接动词原形 save，用 **so as to**（在句末）。
- **排除法**：A/C 后面需完整从句；D 让步。
- **考查知识点**：so as to + 动词原形（句末）。

**第 14 题：答案 B**

- **解析**：句首位置，后面是完整从句 we might not miss...，用 **In order that**。
- **排除法**：A So that 通常不放句首；C/D 后接动词原形但这里是完整从句。
- **考查知识点**：in order that + 从句（可放句首）。

**第 15 题：答案 D**

**第 16 题：答案 B**

- **排除法**：A so that 语义不当（不是"为了下雨"）；C 让步；D because 原因。
- **考查知识点**：in case 表"以防"与 so that 的区分。

**第 18 题：答案 B**

- **解析**："他安静地工作为了不打扰室友"，否定目的用 **so as not to**。
- **排除法**：A in order to 是肯定；C/D 后面需完整从句。
- **考查知识点**：so as not to 表否定目的。

**第 20 题：答案 C**

- **解析**：放在句首，后面是完整从句 they could finish...，用 **In order that**。
- **排除法**：A/B 后接动词原形但这里是完整从句；D 不存在 so that to。
- **考查知识点**：in order that + 从句（句首）。
