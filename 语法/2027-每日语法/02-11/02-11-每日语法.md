# There Be 句型——构成、时态变化与主谓一致（There Be Sentence Pattern）

> 日期：2027 年 02 月 11 日 | 语法分类：句法 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**There be 句型** 是英语中用来表示"某地/某时**存在**某人或某物"的基本句型。它的核心结构是：

> **There + be + 主语 + 地点/时间状语**

注意：句中的 there 不是"那里"的意思，它只是一个**引导词**，真正的主语在 be 动词之后。

---

### 2. 构成形式

#### 2.1 基本结构

| 结构 | 示例 |
|------|------|
| There is + 单数名词/不可数名词 + 地点 | There **is** a book on the desk. |
| There are + 复数名词 + 地点 | There **are** some students in the classroom. |

#### 2.2 否定形式

在 be 动词后加 **not**（通常缩写为 isn't / aren't）。

| 形式 | 示例 |
|------|------|
| There is not (isn't) | There **isn't** any water in the glass. |
| There are not (aren't) | There **aren't** any trees near the house. |

> some 在否定句中通常变为 **any**。

#### 2.3 一般疑问句

将 be 动词提到 there 前面。

| 形式 | 示例 |
|------|------|
| Is there...? | **Is there** a hospital near here? —Yes, there is. / No, there isn't. |
| Are there...? | **Are there** any shops on this street? —Yes, there are. / No, there aren't. |

---

### 3. 时态变化

There be 句型可以有多种时态变化，be 动词要随时态而变。

| 时态 | 结构 | 示例 |
|------|------|------|
| 一般现在时 | There is/are... | There **is** a park near our school. |
| 一般过去时 | There was/were... | There **was** a tall tree here ten years ago. |
| 一般将来时 | There will be... / There is/are going to be... | There **will be** a meeting tomorrow. |
| 现在完成时 | There has/have been... | There **have been** many changes in our city. |

---

### 4. 主谓一致——就近原则

当 there be 句型中有**两个或以上**并列主语时，be 动词的单复数与**最靠近**它的那个主语保持一致。这就是**就近原则（proximity principle）**。

| 示例 | 解析 |
|------|------|
| There **is** a pen and two books on the desk. | a pen（单数）离 be 最近 → is |
| There **are** two books and a pen on the desk. | two books（复数）离 be 最近 → are |

---

### 5. 标志词 / 关键信号

| 标志词/短语 | 说明 |
|------------|------|
| there is / there are | 表存在 |
| there was / there were | 过去存在 |
| there will be / there is going to be | 将来存在 |
| some / any / many / much / a few / a lot of | 常与 there be 搭配 |
| 地点状语（on the desk / in the room / near...） | 通常位于句末 |

---

### 6. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-------------|-------------|----------|
| 1 | There have a book on the desk. | There **is** a book on the desk. | there be 不能用 have 代替 |
| 2 | There are a pen and two books. | There **is** a pen and two books. | 就近原则：a pen 是单数，用 is |
| 3 | There will have a party. | There **will be** a party. | 将来时用 will be，不用 will have |
| 4 | There are going to have a meeting. | There **is going to be** a meeting. | 同上，且 a meeting 单数用 is |
| 5 | Is there some milk? | Is there **any** milk? | 疑问句中 some 变 any |

### 逐题解析

**第 1 题：答案 A**

- **解析**：a big tree 是单数名词，There be 句型用 **is**。
- **排除法**：C/D have/has 不能用于 there be 句型。
- **考查知识点**：基本结构——There is + 单数名词。

**第 2 题：答案 A**

- **解析**：water 是不可数名词，用 **is**。
- **排除法**：B 复数用；C/D 不用于 there be。
- **考查知识点**：基本结构——There is + 不可数名词。

**第 3 题：答案 B**

- **解析**：apples 是复数名词，疑问句形式为 **Are** there...?
- **排除法**：A 单数用；C/D 不用于 there be 句型。
- **考查知识点**：一般疑问句——Are there + 复数？

**第 4 题：答案 B**

- **解析**：就近原则。a pen（单数）最靠近 be 动词，用 **is**。
- **排除法**：A are 不符就近原则；C/D 不用于 there be。
- **考查知识点**：主谓一致——就近原则。

**第 5 题：答案 C**

- **解析**：yesterday 表过去时；many students 是复数，用 **were**。
- **排除法**：A/B 是现在时；D was 用于单数。
- **考查知识点**：时态变化——一般过去时 There were。

**第 6 题：答案 B**

- **解析**：There is going to be... 表将来存在；a football match 是单数，用 **is**。
- **排除法**：A are 用于复数主语；C 结构不完整；D 不用于 there be。
- **考查知识点**：时态变化——一般将来时；易错点 4。

**第 7 题：答案 B**

- **解析**：后文"去买一些"说明没有牛奶了，用否定 **isn't**；milk 不可数名词用 is。
- **排除法**：A 肯定不合语境；C/D 用于复数。
- **考查知识点**：否定形式——There isn't any...

**第 8 题：答案 A**

- **解析**：There be 句型的肯定回答为 **Yes, there is.**
- **排除法**：B/D 主语不对；C 是复数形式。
- **考查知识点**：一般疑问句的简短回答。

**第 9 题：答案 B**

- **解析**：rain 是不可数名词，用单数 be 动词；last year 为过去时，用 **was**。
- **排除法**：A/C 是现在时；D were 用于复数。
- **考查知识点**：时态变化——一般过去时 + 不可数名词。

**第 10 题：答案 C**

- **解析**：There will **be**... 是将来时固定结构，will 后接动词原形 be。
- **排除法**：A/B 不是原形；D 不用于 there be。
- **考查知识点**：时态变化——一般将来时；易错点 3。

**第 11 题：答案 B**

- **解析**：就近原则。two cups of tea（复数）最靠近 be 动词，用 **are**。
- **排除法**：A 单数不合；C/D 不用于 there be。
- **考查知识点**：主谓一致——就近原则。

**第 12 题：答案 A**

- **解析**：time（时间）是不可数名词，用 **is**。There is no time = 没有时间了。
- **排除法**：B 复数用；C/D 不用于 there be。
- **考查知识点**：基本结构——There is + 不可数名词。

**第 13 题：答案 A**

- **解析**：since 2020 表示从过去持续到现在，用**现在完成时**；changes 是复数，用 **have been**。
- **排除法**：B has been 用于单数；C 是一般过去时；D 是一般现在时。
- **考查知识点**：时态变化——现在完成时 There have been。

**第 14 题：答案 B**

- **解析**：就近原则。a book（单数）最靠近 be 动词，用 **is**。
- **排除法**：A are 不符就近原则；C/D 不用于 there be。
- **考查知识点**：主谓一致——就近原则。

**第 15 题：答案 B**

- **解析**：How many students 是复数，用 **are**。
- **排除法**：A 单数用；C/D 不用于 there be。
- **考查知识点**：特殊疑问句中的 there be。

**第 16 题：答案 B**

- **解析**：something 是不定代词，视为单数，用 **is**。
- **排除法**：A 复数用；C/D 不用于 there be。
- **考查知识点**：基本结构——不定代词 + is。

**第 17 题：答案 A**

- **解析**：**There used to be...** 过去曾经有……，used to 后接动词原形 **be**。
- **排除法**：B/C 不是原形；D 不用于 there be。
- **考查知识点**：固定搭配——There used to be...

**第 18 题：答案 B**

- **解析**：little 表否定含义（几乎没有），反意疑问句用**肯定**形式 **is there**；water 不可数名词用 is。
- **排除法**：A isn't there 反义错误；C/D 用 are 不对。
- **考查知识点**：反意疑问句 + little 表否定。

**第 19 题：答案 B**

- **解析**：nothing 是不定代词，视为单数，用 **is**。
- **排除法**：A 复数用；C/D 不用于 there be。
- **考查知识点**：基本结构——不定代词 + is。

**第 20 题：答案 C**

- **解析**：will there **be** 是将来时疑问句，will 后接原形 **be**。
- **排除法**：A/B 不是原形；D been 是过去分词。
- **考查知识点**：将来时疑问句——will there be。
