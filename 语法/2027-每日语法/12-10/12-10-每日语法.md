# 被动语态——一般将来时与现在进行时的被动（Passive Voice — Future & Present Continuous Passive）

> 日期：2027 年 12 月 10 日 | 语法分类：语态 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

被动语态的核心结构是 **be + 过去分词**。不同时态的被动语态通过改变 **be** 的形式来体现。前面已学过一般现在时、一般过去时和含情态动词的被动语态，本节重点学习**一般将来时的被动语态**和**现在进行时的被动语态**，以及被动语态的综合运用。

### 2. 构成形式

| 时态 | 主动语态 | 被动语态 | 示例 |
|------|---------|---------|------|
| 一般现在时 | do/does | am/is/are + done | English is spoken here. |
| 一般过去时 | did | was/were + done | The window was broken yesterday. |
| 一般将来时 | will do | will be + done | The bridge will be built next year. |
| 现在进行时 | am/is/are doing | am/is/are + being + done | The road is being repaired now. |
| 现在完成时 | have/has done | have/has + been + done | The work has been finished. |
| 含情态动词 | can/must/should do | can/must/should + be + done | It must be done at once. |

| 句型 | 一般将来时被动 |
|------|--------------|
| 肯定句 | 主语 + will be + 过去分词 |
| 否定句 | 主语 + will not (won't) be + 过去分词 |
| 疑问句 | Will + 主语 + be + 过去分词? |

| 句型 | 现在进行时被动 |
|------|--------------|
| 肯定句 | 主语 + am/is/are + being + 过去分词 |
| 否定句 | 主语 + am/is/are + not + being + 过去分词 |
| 疑问句 | Am/Is/Are + 主语 + being + 过去分词? |

### 3. 用法与语境

- **用法一**：一般将来时被动——将要被做的事
  - 例：A new hospital will be built in our city.（我们城市将要建一所新医院。）
  - 例：The meeting won't be held tomorrow.（会议明天不会举行。）
  - 例：Will the work be finished on time?（工作能按时完成吗？）

- **用法二**：现在进行时被动——正在被做的事
  - 例：The bridge is being repaired now.（桥现在正在被修。）
  - 例：The students are being interviewed at the moment.（学生们此刻正在被面试。）

- **用法三**：现在完成时被动——已经被做完的事
  - 例：The homework has already been finished.（作业已经完成了。）
  - 例：Many trees have been planted this year.（今年已经种了很多树。）

- **用法四**：主动变被动的步骤
  1. 将原主动句的宾语变为被动句的主语
  2. 将动词变为"be + 过去分词"（be 的形式与时态一致）
  3. 原主动句的主语变为 by + 宾格（可省略）
  - 例：They will hold a meeting tomorrow. → A meeting will be held (by them) tomorrow.

### 4. 标志词 / 关键信号

| 标志词/短语 | 对应时态被动 | 示例 |
|------------|-------------|------|
| tomorrow / next year / soon | 一般将来时被动 | will be done |
| now / at the moment | 现在进行时被动 | is being done |
| already / yet / just | 现在完成时被动 | has been done |
| by + 执行者 | 被动标志 | was built by workers |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | The bridge will built next year. | The bridge will be built next year. | 将来时被动缺 be |
| 2 | The road is being repair now. | The road is being repaired now. | being 后接过去分词不是原形 |
| 3 | The work has finished. | The work has been finished. | 完成时被动缺 been |
| 4 | A new school will be build. | A new school will be built. | build 的过去分词是 built |
| 5 | The book is being reading. | The book is being read. | 被动中用过去分词 read 不是现在分词 |

### 逐题解析

**第 1 题：答案 B**

- **解析**：next year 表将来，bridge 是被建造的，用一般将来时被动 will be built。
- **排除法**：A. 缺 be；C. 一般现在时不合；D. 一般过去时不合。
- **考查知识点**：知识点精讲 → 用法一。

**第 2 题：答案 B**

- **解析**：now 表正在进行，road 正在被修，用现在进行时被动 is being repaired。
- **排除法**：A. 一般现在时被动不表"正在"；C/D 时态不对。
- **考查知识点**：知识点精讲 → 用法二。

**第 3 题：答案 A**

- **解析**：already 提示现在完成时，work 是被完成的，has been finished。
- **排除法**：B. work 用 has；C. being 不对；D. 缺 has。
- **考查知识点**：知识点精讲 → 用法三。

**第 4 题：答案 C**
