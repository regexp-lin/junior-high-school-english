# 祈使句——结构、否定形式与用法（Imperative Sentences — Structure, Negation & Usage）

> 日期：2027 年 11 月 21 日 | 语法分类：句型 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

祈使句（Imperative Sentence）用于表示**命令、请求、建议、警告、禁止**等，句子通常**省略主语 you**，以**动词原形**开头。祈使句的语气可轻可重：加 please 变得更礼貌，加 don't 变成否定形式。

### 2. 构成形式

| 类型 | 结构 | 示例 |
|------|------|------|
| 肯定祈使句 | 动词原形 + ... | Open the door, please. |
| 否定祈使句 | Don't + 动词原形 + ... | Don't be late for school. |
| Let 开头 | Let + 宾语 + 动词原形 | Let me help you. / Let's go. |
| Let 否定 | Let's not + 动词原形 / Don't let + 宾语 + 动词原形 | Let's not talk about it. / Don't let him go. |
| Be 开头 | Be + 形容词/名词 | Be quiet! / Be a good boy. |
| Never 开头 | Never + 动词原形 | Never give up! |

| 句型 | 结构 | 示例 |
|------|------|------|
| 命令 | 动词原形 | Sit down! |
| 请求 | 动词原形 + please | Please come in. |
| 建议 | Let's + 动词原形 | Let's have a rest. |
| 禁止 | Don't / Never + 动词原形 | Don't smoke here. |

### 3. 用法与语境

- **用法一**：表示命令或指示
  - 例：Stand up and answer the question.（站起来回答问题。）
  - 例：Turn off your phone during the exam.（考试期间关掉手机。）

- **用法二**：表示请求或邀请（常加 please）
  - 例：Please pass me the salt.（请把盐递给我。）
  - 例：Come and have dinner with us.（来和我们一起吃晚饭吧。）

- **用法三**：表示建议（Let's 开头）
  - 例：Let's play basketball after school.（放学后我们打篮球吧。）
  - 例：Let's not waste time.（我们不要浪费时间。）

- **用法四**：表示警告或禁止（Don't / Never）
  - 例：Don't touch the hot stove!（不要碰热炉子！）
  - 例：Never speak to me like that again!（别再那样跟我说话！）

### 4. 标志词 / 关键信号

| 标志词/短语 | 说明 | 示例 |
|------------|------|------|
| please | 使祈使句更礼貌 | Please sit down. |
| Don't | 否定祈使句 | Don't run in the hallway. |
| Never | 强烈否定 | Never tell a lie. |
| Let's | 提建议（包含说话者） | Let's go swimming. |
| Let me/him/her | 请求允许 | Let me try. |
| Be | 用于 be 动词开头的祈使句 | Be careful! |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | Don't to be late. | Don't be late. | Don't 后直接接动词原形，不加 to |
| 2 | Not run in the classroom. | Don't run in the classroom. | 否定祈使句用 Don't，不是 Not |
| 3 | Let's go, shall you? | Let's go, shall we? | Let's 的反意疑问用 shall we |
| 4 | Opens the window, please. | Open the window, please. | 祈使句用动词原形，不加 s |
| 5 | Please you sit down. | Please sit down. | 祈使句省略主语 you |

### 逐题解析

**第 1 题：答案 B**

- **解析**：Be quiet! Be 动词原形开头的祈使句。
- **排除法**：A/C 不是原形；D. Being 不对。
- **考查知识点**：知识点精讲 → 构成形式：Be + 形容词。

**第 2 题：答案 C**

- **解析**：否定祈使句用 Don't + 动词原形。
- **排除法**：A. Not 不能直接放句首；B. No 不用于此结构；D. Doesn't 不用于祈使句。
- **考查知识点**：知识点精讲 → 用法四及易错点 2。

**第 3 题：答案 C**

- **解析**：Let's + 动词原形，go for a walk。
- **排除法**：A. to go 不对；B. going 不对；D. goes 不对。
- **考查知识点**：知识点精讲 → 用法三：Let's + 动词原形。

**第 4 题：答案 D**

- **解析**：Please + 动词原形，close the door。
- **排除法**：A/B/C 形式不对。
- **考查知识点**：知识点精讲 → 用法二：请求加 please。

**第 5 题：答案 B**

- **解析**：否定祈使句 Don't be late。
- **排除法**：A. Not be 不对；C. Doesn't be 不存在；D. Not to be 不对。
- **考查知识点**：知识点精讲 → 构成形式及易错点 1：Don't be。

**第 6 题：答案 D**

- **解析**：Don't give up 和 Never give up 都是正确的否定祈使句。
- **排除法**：A. Not 不能单独用。
- **考查知识点**：知识点精讲 → 用法四：Don't 和 Never 都可以。

**第 7 题：答案 A**
