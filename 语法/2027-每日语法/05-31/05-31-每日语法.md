# 结果状语从句——so...that 与 such...that（Adverbial Clauses of Result — so...that & such...that）

> 日期：2027 年 5 月 31 日 | 语法分类：从句 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**结果状语从句**（Adverbial Clause of Result）用来说明主句动作或状态带来的**结果**，常译为“如此……以至于……”。初中最常考的两组结构是 **so...that...** 与 **such...that...**。二者意思相近，但**中心词不同**：**so** 后面常接**形容词或副词**；**such** 后面要接**名词（或名词短语）**。

### 2. 构成形式

| 结构 | 中心词 | 公式 | 示例 |
|------|--------|------|------|
| **so...that** | 形容词/副词 | **so + adj./adv. + that 从句** | He ran **so fast** **that** I couldn't catch him. |
| **so...that** | 数量修饰 + 名词 | **so + many/few + 复数名词 + that** | There were **so many people** **that** we had to wait. |
| **so...that** | 数量修饰 + 名词 | **so + much/little + 不可数名词 + that** | There was **so much noise** **that** I couldn't sleep. |
| **such...that** | 名词短语 | **such + a/an + adj. + 单数可数名词 + that** | It was **such a cold day** **that** we stayed home. |
| **such...that** | 名词短语 | **such + adj. + 复数/不可数名词 + that** | They are **such helpful teachers** **that** students love them. |
| **同义替换（拔高）** | 名词短语 | **so + adj. + a/an + 单数名词** = **such a/an + adj. + 单数名词** | **so nice a day** = **such a nice day** |

### 3. 用法与语境

- **用法一：so + 形容词/副词 + that（强调性质或程度）**
  - 例：The math problem was **so difficult** **that** half of the class gave up.（这道数学题太难了，以至于班里一半人都放弃了。）
  - 例：Tom spoke **so clearly** **that** everyone understood him.（汤姆说得如此清楚，以至于大家都听懂了。）

- **用法二：so many / so few + 复数名词 + that（强调数量多/少带来的结果）**
  - 例：There were **so few seats** **that** some parents had to stand.（座位太少了，以至于有些家长只能站着。）

- **用法三：so much / so little + 不可数名词 + that（强调量多/少）**
  - 例：We had **so little time** **that** we skipped lunch.（我们时间太少了，以至于没吃午饭。）

- **用法四：such + 名词短语 + that（强调“这样的人/事/物”）**
  - 例：It was **such an exciting match** **that** we forgot to go home.（那场比赛太精彩了，以至于我们都忘了回家。）
  - 例：This is **such useful advice** **that** I'll write it in my notebook.（这是如此有用的建议，我要记在笔记本上。）

### 4. 标志词 / 关键信号

| 标志词/短语 | 说明 | 示例 |
|------------|------|------|
| **so + adj./adv.** | 后面紧跟形容词或副词 | **so tired that** he fell asleep |
| **such + a/an + n.** | such 后接带冠词的名词短语 | **such a good book that** … |
| **so many / so much** | 强调数量导致结果 | **so much homework that** … |
| **that 从句表结果** | that 后常是完整句（主谓结构） | … **that** we couldn't move |
| **so...that 与 very...** | very 不引导结果从句；so/such 才与 that 搭配 | ❌ very cold that → ✅ **so cold that** |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | **such** tired **that** he stopped | **so** tired **that** he stopped | **tired** 是形容词，用 **so**，不用 **such**。 |
| 2 | **so** a good student **that** … | **such** a good student **that** … | **a good student** 是名词短语，用 **such**。 |
| 3 | **such** difficult problem **that** … | **such a** difficult problem **that** … | 单数可数名词前一般要有 **a/an**。 |
| 4 | **so useful advice that** … | **such useful advice that** … | **advice** 不可数，用 **such + adj. + 不可数名词**。 |
| 5 | **so many noise that** … | **so much noise that** … | **noise** 不可数，用 **much**，不用 **many**。 |

### 逐题解析

**第 1 题：答案 A**

- **解析**：**interesting** 是形容词，表示“如此有趣以至于……”要用 **so + 形容词 + that**。对应知识点精讲 → 用法一。
- **排除法**：**such** 后接名词短语，不接单个形容词 **interesting**；**very/too** 不能与 **that** 引导结果从句这样搭配（**too...to...** 表“太……而不能”，结构不同）。
- **考查知识点**：**so + adj. + that** 表结果。

**第 2 题：答案 B**

- **解析**：**heavy rain** 是名词短语（不可数），用 **such + 形容词 + 不可数名词 + that**。对应知识点精讲 → 用法四。
- **排除法**：**so** 后不能直接接名词短语 **heavy rain**；**rain** 不可数，不能加 **a**。
- **考查知识点**：**such + adj. + 不可数名词 + that**。

**第 3 题：答案 C**

- **解析**：**fast** 是副词，修饰 **spoke**，用 **so fast that**。对应知识点精讲 → 用法一。
- **排除法**：**such** 不直接修饰副词；**very...that**、**too...that** 不是正确搭配。
- **考查知识点**：**so + adv. + that**。

**第 4 题：答案 D**

- **解析**：**interesting** 以元音音素开头，单数可数名词 **book** 前用 **an**，完整结构为 **such an interesting book that...**。对应知识点精讲 → 用法四。
- **排除法**：**so...that** 若修饰名词需用 **so interesting a book**，不能写成 **so interesting book**；**such...that** 缺冠词；**such a interesting** 冠词与元音开头形容词不搭配，应为 **an**。
- **考查知识点**：**such an + adj. + 单数可数名词 + that**。

**第 5 题：答案 A**

- **解析**：**hungry** 是形容词，用 **so hungry that**。对应易错点：形容词前用 **so** 不用 **such**。
- **排除法**：**such** 不直接修饰形容词；**too...to** 后接不定式，不接 **that** 从句。
- **考查知识点**：**so + adj. + that**。

**第 6 题：答案 B**

- **解析**：**mistakes** 为复数可数名词，表“如此多的错误”用 **so many...that**。对应知识点精讲 → 用法二。
- **排除法**：**much** 修饰不可数名词；英语中一般不说 **such many**。
- **考查知识点**：**so many + 复数名词 + that**。

**第 7 题：答案 C**

- **解析**：**tip** 为单数可数名词，**a useful tip** 前用 **such a...that**。对应知识点精讲 → 用法四。
- **排除法**：不能说 **so a useful tip**；单独 **such...that** 后缺冠词，不接好 **useful tip**。
- **考查知识点**：**such a/an + adj. + 单数名词 + that**。

**第 8 题：答案 D**

- **解析**：**loud** 是形容词，用 **so loud that**。对应知识点精讲 → 用法一。
- **排除法**：**such** 后需名词；**too...to** 结构不符（后面是 **that** 从句）。
- **考查知识点**：**so + adj. + that**。

**第 9 题：答案 A**

- **解析**：**honest** 以元音音素开头，用 **such an honest boy that**。对应知识点精讲 → 用法四。
- **排除法**：**so** 后接形容词时不再插入 **a/an + 名词** 这种名词短语结构；无 **so an** 搭配。
- **考查知识点**：**such an + adj. + 单数可数名词 + that**。

**第 10 题：答案 B**

- **解析**：**time** 不可数，“如此少的时间”用 **so little time that**。对应知识点精讲 → 用法三。
- **排除法**：**such little** 不是初中常考的正确搭配；单独 **so...that** 缺 **little** 则无法表达“少”。
- **考查知识点**：**so little + 不可数名词 + that**。

**第 11 题：答案 C**

- **解析**：**crowded** 是形容词，用 **so crowded that**。对应知识点精讲 → 用法一（拔高语境）。
- **排除法**：**such** 后需名词；**too...to** 不接 **that**。
- **考查知识点**：**so + adj. + that**。

**第 12 题：答案 D**

- **解析**：需要 **such a cold morning that...** 两个空一起完成“如此一个寒冷的早晨以至于……”。对应知识点精讲 → 构成形式。
- **排除法**：**such a** 单独无法接 **that**；**so a** 不规范。
- **考查知识点**：**such a + adj. + 单数名词 + that**。

**第 13 题：答案 A**

- **解析**：**milk** 不可数，“如此少”用 **so little milk that**。对应易错点辨析表第 5 条相关（可数与不可数）。
- **排除法**：**few** 修饰可数复数；**such little** 非标准结构。
- **考查知识点**：**so little + 不可数名词 + that**。

**第 14 题：答案 B**

- **解析**：**boring** 修饰 **movie**（可用形容词说明事物性质），用 **so boring that**。对应知识点精讲 → 用法一。
- **排除法**：**such boring** 需接名词（如 **such a boring movie**），题干结构是 **was ______ boring**；**very/too** 与 **that** 不这样搭配。
- **考查知识点**：**so + adj. + that**。

**第 15 题：答案 C**

- **解析**：**kind volunteers** 为复数名词短语，用 **such kind volunteers that**。对应知识点精讲 → 用法四（复数）。
- **排除法**：**so** 后不能直接接 **kind volunteers** 这样的名词短语；**volunteers** 复数不能加 **a**。
- **考查知识点**：**such + adj. + 复数名词 + that**。

**第 16 题：答案 D**

- **解析**：**mistakes** 可数复数，用 **so many mistakes that**。对应易错点：不可数用 **much**，可数复数用 **many**。
- **排除法**：**much** 不接可数复数；**such many / such much** 不成立。
- **考查知识点**：**so many + 复数名词 + that**。

**第 17 题：答案 A**

- **解析**：**easy** 是形容词，**sounds so easy that...** 表示“听起来如此简单以至于……（很多学生因粗心做错）”。对应知识点精讲 → 用法一（拔高：系动词 + 形容词）。
- **排除法**：**such** 不直接修饰形容词 **easy**；**very/too** 不与 **that** 结果从句这样连用。
- **考查知识点**：**so + adj. + that**。

**第 18 题：答案 B**

- **解析**：**surprise** 此处为可数名词（一件令人惊喜的事），**such a nice surprise that**。对应知识点精讲 → 用法四。
- **排除法**：单数可数名词需 **a**；**such** 单独或 **so a** 均不完整。
- **考查知识点**：**such a + adj. + 可数名词 + that**。

**第 19 题：答案 C**

- **解析**：**noise** 不可数，用 **so much noise that**。对应易错点辨析表第 5 条。
- **排除法**：**many** 不接不可数名词；**such many/much** 错误。
- **考查知识点**：**so much + 不可数名词 + that**。

**第 20 题：答案 D**

- **解析**：**dangerous** 是形容词，**so dangerous that**。对应知识点精讲 → 用法一（拔高：综合语境）。
- **排除法**：**such dangerous** 需接名词短语，题干为 **was ______ dangerous**；**very/too** 不符合 **that** 结果从句结构。
- **考查知识点**：**so + adj. + that**。
