# 情态动词——can / could / be able to 表能力（Modal Verbs — *can / could / be able to* for Ability）

> 日期：2027 年 8 月 16 日 | 语法分类：特殊结构 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**情态动词（modal verbs）**中的 **can**、**could** 以及短语 **be able to** 都可以表示“**能力（ability）**”或与之相关的“**可能做到/做不到**”。其中 **can** 最常用，语气直接；**could** 常表示**过去的一般能力**或**更委婉、更客气**的说法；**be able to** 可以配合 **will、have、be** 等构成**多种时态**，也常用来表示**某次具体情境中“终于做到/没能做到”**。

### 2. 构成形式

**说明**：下表中 **V** 表示动词原形；**be** 随主语和人称变化（**am / is / are / was / were**）。

| 句型 | 结构（structure） | 示例 |
|------|------------------|------|
| **肯定句（affirmative）** | 主语 + **can** + **V** | *She **can** play the piano.* |
| | 主语 + **could** + **V** | *He **could** swim when he was six.* |
| | 主语 + **am/is/are able to** + **V** | *We **are able to** finish it today.* |
| **否定句（negative）** | 主语 + **can't / cannot** + **V** | *I **can't** drive yet.* |
| | 主语 + **couldn't** + **V** | *They **couldn't** hear us clearly.* |
| | 主语 + **am not / isn't / aren't able to** + **V** | *He **isn't able to** come tonight.* |
| **一般疑问句（yes/no question）** | **Can** + 主语 + **V** …? | ***Can** you help me?* |
| | **Could** + 主语 + **V** …? | ***Could** I sit here?* |
| | **Be** + 主语 + **able to** + **V** …? | ***Are** they **able to** leave now?* |
| **特殊疑问句（Wh- question）** | 疑问词 + **can/could** + 主语 + **V** …? | *What **can** this robot do?* |
| | 疑问词 + **be** + 主语 + **able to** + **V** …? | *When **will** you **be able to** start?* |

**补充**：**be able to** 的常见时态扩展——**will be able to**（将来能够）、**was/were able to**（过去某次能够做到）、**have/has been able to**（从过去某时到现在一直能够）等。

### 3. 用法与语境

- **用法一：** **can** 表示**现在的能力、学会的技能或一般情况下的“会做”**（present ability / general ability）。  
  - 例：*My cousin **can** cook simple meals now.*（我表哥现在会做简单的饭菜了。）

- **用法二：** **could** 表示**过去的一般能力**（past general ability），相当于“过去常常能够 / 过去会做”。  
  - 例：*Grandpa **could** run very fast in his twenties.*（爷爷二十多岁时跑得很快。）

- **用法三：** **be able to** 用于**各时态**，强调“具备做某事的能力或条件”；**was/were able to** 常表示**过去某一具体场合中“终于做到/成功做到”**（single successful action），与只表示“过去会”的 **could** 有细微差别。  
  - 例：*I **will be able to** visit you next Sunday.*（下周日我就能去看你了。）  
  - 例：*The door was stuck, but Dad **was able to** open it.*（门卡住了，但爸爸还是把它打开了。）

- **用法四：** **can** 还可表示**请求许可**（asking for permission），多用于口语，相当于“可以……吗？”。  
  - 例：***Can** I use your pen?*（我可以用你的钢笔吗？）

- **用法五：** **could** 在疑问句中常比 **can** **更委婉、更客气**（more polite），但答句中说明**现在的真实能力或许可**时，仍多用 **can**。  
  - 例：*—**Could** you help me carry this box? —Sure, I **can**.*（—你能帮我搬这个箱子吗？—当然，可以。）

### 4. 标志词 / 关键信号

| 标志词 / 短语 | 说明 | 示例 |
|--------------|------|------|
| **now / these days / at the moment** | 与**现在能力**呼应，常用 **can** | *I **can** speak more clearly **now**.* |
| **when I was… / at the age of… / in 2020** | 表**过去习惯能力**，常用 **could** | *She **could** draw well **when she was little**.* |
| **yesterday / last night / just now** | 表**过去具体情境**，能力+结果常用 **was/were able to** 或 **couldn't** | *We **were able to** catch the last bus **last night**.* |
| **tomorrow / next week / soon** | **将来能够**，常用 **will be able to** | *You'll **be able to** join the club **next term**.* |
| **for… / since…**（与完成时连用） | 与 **have/has been able to** 搭配表“从…起一直能” | *He **has been able to** ride to school **since** Grade 8.* |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *I **could** open the door*（强调刚才经过一番努力终于打开） | *I **was able to** open the door.* | 表**过去某次具体成功**，肯定句优先用 **was/were able to** / **managed to**，不用单独 **could** 表“终于做到”。 |
| 2 | *She **can** to swim.* | *She **can swim**.* / *She **is able to** swim.* | **can** 后接**动词原形**，不能加 **to**；**be able to** 才带 **to**。 |
| 3 | *Can you to help me?* | *Can you **help** me?* | 疑问句中 **can** 后也是**原形**，不能插入 **to**。 |
| 4 | *I will can finish it tomorrow.* | *I **will be able to** finish it tomorrow.* | **can** 一般**没有** *will can* 这类将来形式，将来多用 **will be able to**。 |
| 5 | *Could you help me? —Yes, I could.*（强调“我现在可以帮”） | *—Could you help me? —Yes, I **can**.* | 回应**现在的能力或许可**时，用 **can** 更自然；**could** 多用于提出请求的委婉语气。 |

## 二、专项选择题（共 20 题）

**说明**：请选出每题的最佳答案。第 1—10 题为中等难度，第 11—20 题为拔高题。

**1.** After months of practice, Tina ________ sing this English song without looking at the lyrics.  
A. can　　B. could　　C. must　　D. should

**2.** When Uncle Li was young, he ________ lift heavy boxes in the factory.  
A. can　　B. could　　C. may　　D. must

**3.** Our head teacher says we ________ use the new reading room as soon as it opens next month.  
A. can　　B. could　　C. will be able to　　D. should

**4.** The lift stopped suddenly, but the engineer ________ fix it in ten minutes.  
A. can　　B. could　　C. should　　D. was able to

**5.** ________ I sit next to the window? I feel a little sick on the bus.  
A. Can　　B. Must　　C. Should　　D. Need

**6.** Excuse me, ________ you tell me how to get to the City Library?  
A. can　　B. could　　C. must　　D. should

**7.** The baby is only ten months old, so he ________ walk alone yet.  
A. can　　B. could　　C. can't　　D. must

**8.** ________ your cousin play chess well?  
A. Does　　B. Is　　C. Has　　D. Can

**9.** With these clear notes, I ________ understand the grammar rules much better now.  
A. can　　B. could　　C. should　　D. must

**10.** In primary school, Lucy ________ run 100 meters in 16 seconds.  
A. can　　B. could　　C. may　　D. must

**11.** The math problem looked hard, but in the end Andy ________ work it out by himself.  
A. can't　　B. couldn't　　C. was able to　　D. isn't able to

**12.** By next winter vacation, Jim ________ ski on the real snow in Harbin with his cousins.  
A. can　　B. could　　C. is able to　　D. will be able to

**13.** I tried the key several times, but I ________ open that old door yesterday.  
A. couldn't　　B. can't　　C. mustn't　　D. shouldn't

**14.** After hard training for a month, the team ________ win the match in the last minute.  
A. can　　B. was able to　　C. could　　D. should

**15.** My phone was off, so I ________ answer your call last night. Sorry!  
A. needn't　　B. mustn't　　C. couldn't　　D. shouldn't

**16.** ________ I borrow your umbrella? It looks like it's going to rain.  
A. Must　　B. Should　　C. May　　D. Could

**17.** How long ________ your sister ________ play the violin?  
A. has; been able to　　B. is; able to　　C. does; able to　　D. did; able to

**18.** Dolphins ________ use sounds to find their way under the water.  
A. may　　B. can　　C. must　　D. should

**19.** Students ________ raise their hands first if they want to say something in class.  
A. must　　B. should　　C. can　　D. need

**20.** I'm busy this Saturday, but I ________ help you clean the classroom on Sunday morning.  
A. can　　B. could　　C. may　　D. will be able to

### 逐题解析

**第 1 题：答案 A**

- **解析**：句中 **After months of practice** 与 **now** 的语境呼应，表示**现在已经具备的能力**，用 **can**。
- **排除法**：**could** 偏重过去或委婉，不贴合“练了几个月后的现在”；**must/should** 表义务、建议，不表能力。
- **考查知识点**：知识点精讲 → **用法一：can 表现在能力**。

**第 2 题：答案 B**

- **解析**：**When Uncle Li was young** 表示**过去的一般能力**，“年轻时能做到”，用 **could**。
- **排除法**：**can** 表现在；**may** 表许可或推测；**must** 表必须或肯定推测，均不表“过去会/能”。
- **考查知识点**：知识点精讲 → **用法二：could 表过去能力**。

**第 3 题：答案 C**

- **解析**：**as soon as it opens next month** 指向**将来**，且强调“条件具备后就能够”，用 **will be able to**；**can** 一般不与 **will** 连用构成将来能力（易错点 4）。
- **排除法**：**can/could** 不贴合明确的“下月开放后才行”；**should** 表“应该”，不是能力结构。
- **考查知识点**：知识点精讲 → **用法三：be able to 各时态（will be able to）**；易错点 4。

**第 4 题：答案 D**

- **解析**：电梯突然停运是**过去具体情境**，强调工程师**在那十分钟内成功修好**，用 **was able to** 比单纯 **could** 更突出“**终于做到**”（易错点 1）。
- **排除法**：仅用 **can** 不体现过去时；**could** 表过去一般能力，不如 **was able to** 贴切“这次成功解决”。
- **考查知识点**：知识点精讲 → **用法三：was/were able to 表过去具体成功**；易错点 1。

**第 5 题：答案 A**

- **解析**：**请求许可**“我可以坐窗边吗？”，口语中常用 **Can I…?**。
- **排除法**：**Must/Should/Need** 不用于此许可请求结构。
- **考查知识点**：知识点精讲 → **用法四：can 表请求许可**。

**第 6 题：答案 B**

- **解析**：向陌生人问路，用 **Could you tell me…?** 比 **Can you…?** **更委婉、更客气**（用法五）。
- **排除法**：**could** 在此不表过去能力，而是礼貌语气；**must/should** 不符合问路习惯。
- **考查知识点**：知识点精讲 → **用法五：could 比 can 更客气**。

**第 7 题：答案 C**

- **解析**：**only ten months old** 与 **yet** 提示**现在还不会**，否定能力用 **can't**。
- **排除法**：**can** 与事实矛盾；**could** 不用于陈述婴儿“当前不会”；**must** 表推测或必须，不表能力否定。
- **考查知识点**：知识点精讲 → **can** 表现在能力及否定形式 **can't**。

**第 8 题：答案 D**

- **解析**：询问**是否会下棋**，用 **Can your cousin play chess…?** **can** 表能力。
- **排除法**：**Does** 后需原形但侧重习惯动作而非“会不会”；**Is** 不能与 **play** 这样搭配表能力；**Has** 结构错误。
- **考查知识点**：构成形式 → **一般疑问句：Can + 主语 + 动词原形**。

**第 9 题：答案 A**

- **解析**：**now** 与 **much better** 说明**现在的理解能力**，用 **can**。
- **排除法**：**could** 易误解为过去或虚拟；**should/must** 不表“能够看懂”。
- **考查知识点**：标志词 **now** + **用法一：can 表现在能力**。

**第 10 题：答案 B**

- **解析**：**In primary school** 表示**过去阶段的一般能力**，“那时能跑……”，用 **could**。
- **排除法**：**can** 表现在；**may/must** 不表“过去会跑”的能力义。
- **考查知识点**：用法二 + 标志词 **In primary school**。

**第 11 题：答案 C**

- **解析**：**looked hard** 与 **in the end** 强调**经过一番努力终于做出来**，肯定句用 **was able to**（易错点 1）。
- **排除法**：**can't/isn't able to** 时态与句意相反；**couldn't** 表示没做到，与 **work it out** 矛盾。
- **考查知识点**：用法三；易错点 1（过去具体成功用 **was able to**）。

**第 12 题：答案 D**

- **解析**：**By next winter vacation** 为**将来时间点**，表示“到那时将能够”，用 **will be able to**。
- **排除法**：**can/could** 不用于该将来结构；**is able to** 表现在，与 **next winter** 不符。
- **考查知识点**：用法三；标志词 **By next…**。

**第 13 题：答案 A**

- **解析**：**yesterday** 说明过去；**tried… but** 表示**做不到**，用过去式 **couldn't**。
- **排除法**：**can't** 表现在；**mustn't** 表禁止；**shouldn't** 表不该，均不表“没能力/没打开”。
- **考查知识点**：**couldn't** 表过去具体情境中的“不能/没能”（与 **wasn't able to** 在此类否定中常可互通）；标志词 **yesterday**。

**第 14 题：答案 B**

- **解析**：**After hard training** + **in the last minute** 描写**过去比赛中最终做到**，用 **was able to** 突出**成功结果**。
- **排除法**：**can** 表现在；单独 **could** 表“过去能赢”较泛，不如 **was able to** 贴合“那场终于赢”；**should** 不表能力实现。
- **考查知识点**：用法三；易错点 1。

**第 15 题：答案 C**

- **解析**：手机关机导致**昨晚没能接电话**，表过去**不能**，用 **couldn't**。
- **排除法**：**needn't** 表不必；**mustn't** 表禁止；**shouldn't** 表不该，均不符合“接不了电话”的原因义。
- **考查知识点**：过去情境中的能力/可能性否定；标志词 **last night**。

**第 16 题：答案 D**

- **解析**：借伞是**委婉请求许可**，**Could I borrow…?** 比 **Can I…?** 更礼貌（用法五）。
- **排除法**：**Must/Should** 不用于此类借物请求；**May** 也可表许可，但本题最佳为更常见的委婉结构 **Could I…**。
- **考查知识点**：用法四、五结合（许可 + 客气）。

**第 17 题：答案 A**

- **解析**：**How long** 与“从过去持续到现在”搭配，常用 **have/has been able to**。
- **排除法**：**is able to** 不可直接与 **how long** 表持续段；**does/did able** 结构错误。
- **考查知识点**：用法三 **have/has been able to**；标志词 **How long… / since…**。

**第 18 题：答案 B**

- **解析**：说明海豚的**客观能力/本能**，用 **can** 表“能够、会”。
- **排除法**：**may** 偏许可或不确定推测；**must** 过强；**should** 表建议，不表物种能力。
- **考查知识点**：用法一 **can** 表一般能力（客观事实）。

**第 19 题：答案 C**

- **解析**：课堂规则：**可以**先举手再发言，表**允许**，用 **can**（用法四延伸：规定中的许可）。
- **排除法**：**must** 过强（题意为“想发言可以先举手”，非强制）；**should** 偏建议；**need** 表需要，不贴切。
- **考查知识点**：**can** 表许可（规则语境）。

**第 20 题：答案 D**

- **解析**：**this Saturday** 忙、**Sunday morning** 将有空，表**将来能够帮忙**，用 **will be able to**。
- **排除法**：**can/could/may** 虽可表许可，但句中强调“下周日那时将具备条件”，**will be able to** 最贴合“时间安排后的能够”。
- **考查知识点**：用法三 **will be able to**；将来时间与能力结合。
