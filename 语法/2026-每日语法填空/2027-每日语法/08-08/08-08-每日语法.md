# 定语从句——关系代词 who / which / that 的基本用法（Attributive Clauses — Relative Pronouns who / which / that）

> 日期：2027 年 8 月 8 日 | 语法分类：从句 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**定语从句（attributive clause）**是用来修饰名词或代词、相当于形容词的从句，通常紧跟在被修饰词之后。被修饰的名词或代词叫**先行词（antecedent）**；引导从句、同时连接主从句并在从句中充当成分的词叫**关系词（relative word）**。本课重点学习关系代词 **who、which、that**：它们既代替先行词，又在从句中作主语、宾语等成分。

### 2. 构成形式

| 先行词（antecedent） | 常用关系代词（relative pronouns） | 说明 |
| :--- | :--- | :--- |
| 指人 | **who / that** | *who* 更常强调“人”；*that* 可通用 |
| 指物 | **which / that** | 多数情况下两者可互换 |
| 指人或物（泛指） | **that** | 在不少固定结构中更常用 *that* |

补充：先行词若**同时涉及人和物**，关系代词一般用 **that**。

### 3. 用法与语境

- **who**：先行词是**人**，在从句中常作**主语**，也可作**宾语**（口语或非正式文体中常见；正式文体宾格可用 *whom*，初中阶段重点掌握 *who*）（对应精讲 2）。
- **which**：先行词是**物**（或动物等当作“事物”描述时），在从句中可作**主语或宾语**（对应精讲 2）。
- **that**：既可指**人**也可指**物**，在从句中作主语或宾语；作宾语时口语中常可省略（对应精讲 2）。
- **通常只用 that、不用 which** 的常见情况：先行词被**最高级（superlative）**、**序数词（ordinal number）**、**all、every、no、the only、the very** 等修饰时，或先行词为 **everything、something、anything、nothing** 等不定代词时（对应精讲 2、本条）。
- **通常只用 which、不用 that** 的常见情况：**介词 + 关系代词**结构（如 *from which*）中只能用 *which*；**非限制性定语从句（non-restrictive attributive clause）**（逗号隔开、补充说明）指物时一般用 *which*，**不用 that**（对应精讲 2、本条）。

### 4. 标志词 / 关键信号

- 名词后出现**完整主谓结构**且该结构修饰前面的名词 → 多为定语从句。
- 先行词是**人**且从句缺**主语** → 优先考虑 **who / that**。
- 先行词是**物**且从句缺**主语 / 宾语** → 优先考虑 **which / that**。
- 出现 **the first / the best / the only / all / every / no / the very** 等 → 常考 **that**。
- **逗号**隔开从句、且从句仅作补充说明 → 指物用 **which**；指人常用 **who**。
- 句中出现 **in / on / from / with + ______** 且空格在**介词后** → 常考 **which**。

### 5. 易错点与辨析

1. **which 与 who 混用**：先行词是人却误用 *which*，或先行词是物却误用 *who*（易混点 1）。
2. **that 的“必选”语境记不住**：遇到最高级、序数词、*all / every / no / the only / the very*、复合不定代词作先行词等，仍选 *which*（易混点 2）。
3. **非限制性定语从句误用 that**：逗号后补充说明的从句，指物时应用 *which*，不用 *that*（易混点 3）。
4. **介词后误用 that**：*in which、from which* 等结构中不能用 *that*（易混点 4）。
5. **what 与关系代词混淆**：*what* 引导名词性从句，**不能**引导定语从句修饰前面的先行词（易混点 5）。

### 逐题解析

**1. 答案：A**  
**解析**：先行词 *the man* 指人，从句缺主语 *The man is standing there*，用关系代词 **who**。  
**排除法**：B *which* 修饰物；C *what* 不引导定语从句；D *where* 表地点，在从句中不作主语指人。  
**考查知识点**：对应**三、用法与语境**第一条与**二、构成形式**（先行词是人 → who / that）。

**2. 答案：B**  
**解析**：先行词 *the story* 指物，从句宾语为 *the story*，用 **which**（*that* 亦可，但选项未给出）。  
**排除法**：A、C 用于指人；D *whose* 表所属关系，与句意不符。  
**考查知识点**：对应**三、用法与语境**第二条与**二、构成形式**（先行词是物 → which / that）。

**3. 答案：C**  
**解析**：先行词被最高级 *the funniest* 修饰，关系代词常用 **that**。  
**排除法**：A *which* 在此类中考高频考点中不如 *that* 典型；B *who* 指人；D *where* 不作宾语指代 *joke*。  
**考查知识点**：对应**三、用法与语境**第四条（最高级修饰 → 常用 that）与**五、易错点 2**。

**4. 答案：D**  
**解析**：先行词被序数词 *the first* 修饰，关系代词用 **that** 更规范。  
**排除法**：A *who* 虽可指人，但序数词结构下优先选 *that*；B、C 与先行词 *boy* 及从句主语需求不符。  
**考查知识点**：对应**三、用法与语境**第四条（序数词修饰 → 常用 that）。

**5. 答案：A**  
**解析**：先行词 *the girl* 指人，从句缺主语，用 **who**。  
**排除法**：B *which* 修饰物；C *what* 不用于定语从句；D *whose* 表“谁的”，语义不符。  
**考查知识点**：对应**三、用法与语境**第一条（who 修饰人，作主语）。

**6. 答案：B**  
**解析**：先行词 *the violin* 指物，从句中作 *playing* 的宾语，用 **that**（*which* 亦可，选项中 *that* 正确）。  
**排除法**：A、C 指人；D *whose* 表所属，与结构不符。  
**考查知识点**：对应**三、用法与语境**第三条（that 可指物，作宾语）。

**7. 答案：C**  
**解析**：先行词为不定代词 *Everything*，关系代词一般用 **that**。  
**排除法**：A *which* 在此类先行词下不如 *that* 常见；B *who* 指人；D *what* 不能引导定语从句。  
**考查知识点**：对应**三、用法与语境**第四条（everything 等 → 常用 that）与**五、易错点 2**。

**8. 答案：D**  
**解析**：逗号后为**非限制性定语从句**，先行词 *my cousin* 指人，用 **who**；非限制性从句指人一般不用 *that*。  
**排除法**：A *which* 指物；B *that* 不宜用于该非限制性结构；C *whose* 语义不符。  
**考查知识点**：对应**三、用法与语境**第五条（非限制性定语从句指人用 who）。

**9. 答案：A**  
**解析**：先行词 *the movie* 指物，从句中作 *watched* 的宾语，用 **that**（*which* 亦可）。  
**排除法**：B、C 指人；D *whose* 表所属，不符。  
**考查知识点**：对应**三、用法与语境**第二、三条（物作先行词；that 作宾语）。

**10. 答案：B**  
**解析**：先行词 *the students* 指人，从句缺主语，用 **who**。  
**排除法**：A *which* 修饰物；C、D 在从句中不作主语（*whose* 表所属；*whom* 作宾语）。  
**考查知识点**：对应**三、用法与语境**第一条（who 修饰人作主语）与**五、易错点 1**。

**11. 答案：C**

**排除法**：A *who* 在此类标志词下不如 *that* 贴合考点；B *which* 指物；D *whom* 作宾语，而从句缺主语。  
**考查知识点**：对应**三、用法与语境**第四条（the only → 常用 that）。

**12. 答案：D**

**13. 答案：A**

**排除法**：B、C 用于指人；D *whose* 不符句意。  
**考查知识点**：对应**三、用法与语境**第二条（which 修饰物/事物化先行词）与**五、易错点 1**。

**14. 答案：B**

**15. 答案：C**
