# 被动语态——含情态动词的被动语态（Passive Voice — with Modal Verbs）

> 日期：2027 年 11 月 23 日 | 语法分类：语态 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

含情态动词的被动语态是指在被动语态中加入情态动词（can、may、must、should、could、will、would 等），用于表达"能被……"、"必须被……"、"应该被……"等含义。其结构为 **"情态动词 + be + 过去分词"**，是被动语态中最重要的拓展考点之一。

### 2. 构成形式

| 句型 | 结构 | 示例 |
|------|------|------|
| 肯定句 | 主语 + 情态动词 + be + 过去分词 | The work must be finished today. |
| 否定句 | 主语 + 情态动词 + not + be + 过去分词 | The door can't be opened. |
| 一般疑问句 | 情态动词 + 主语 + be + 过去分词? | Can the problem be solved? |
| 特殊疑问句 | 疑问词 + 情态动词 + 主语 + be + 过去分词? | When should the report be handed in? |

| 情态动词 | 主动语态 | 被动语态 |
|---------|---------|---------|
| can | We can see the stars. | The stars can be seen. |
| must | You must finish the work. | The work must be finished. |
| should | We should protect the environment. | The environment should be protected. |
| may | You may use the computer. | The computer may be used. |
| will | They will build a new bridge. | A new bridge will be built. |

### 3. 用法与语境

- **用法一**：can/could + be + 过去分词，表"能够被……"
  - 例：This problem can be solved easily.（这个问题能被轻松解决。）
  - 例：The old building could be seen from far away.（从远处就能看到那座旧建筑。）

- **用法二**：must + be + 过去分词，表"必须被……"
  - 例：The homework must be handed in on time.（作业必须按时上交。）
  - 例：The rules must be followed by everyone.（规则必须被每个人遵守。）

- **用法三**：should + be + 过去分词，表"应该被……"
  - 例：Children should be taken good care of.（孩子们应该被好好照顾。）
  - 例：The old people should be respected.（老人应该被尊重。）

- **用法四**：will + be + 过去分词，表"将会被……"
  - 例：A new school will be built next year.（明年将建一所新学校。）
  - 例：The meeting will be held tomorrow.（会议将于明天举行。）

### 4. 标志词 / 关键信号

| 标志词/短语 | 说明 | 示例 |
|------------|------|------|
| can/can't + be + V-ed | 能/不能被…… | It can't be moved. |
| must + be + V-ed | 必须被…… | It must be done now. |
| should + be + V-ed | 应该被…… | It should be cleaned. |
| may + be + V-ed | 可以被…… | It may be used. |
| will + be + V-ed | 将被…… | It will be repaired soon. |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | The work must finished today. | The work must be finished today. | 情态动词被动语态不能省略 be |
| 2 | Trees should planted in spring. | Trees should be planted in spring. | 缺少 be |
| 3 | The window can be open. | The window can be opened. | open 需用过去分词 opened |
| 4 | Must the homework be finish today? | Must the homework be finished today? | finish 需用过去分词 finished |
| 5 | The problem can solved by us. | The problem can be solved by us. | 缺少 be |

### 逐题解析

**第 1 题：答案 B**

- **解析**：must + be + 过去分词，must be finished"必须被完成"。
- **排除法**：A. finish 缺 be；C. finished 缺 be 和 must 搭配不对；D. be finishing 不是被动。
- **考查知识点**：知识点精讲 → 用法二：must + be + V-ed。

**第 2 题：答案 A**

- **解析**：should + be + 过去分词，should be planted"应该被种植"。
- **排除法**：B/C 缺 be；D. be planting 不是被动。
- **考查知识点**：知识点精讲 → 用法三：should + be + V-ed。
