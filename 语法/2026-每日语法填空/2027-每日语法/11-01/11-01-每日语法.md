# there be 句型——与各种时态的搭配（There Be — with Various Tenses）

> 日期：2027 年 11 月 1 日 | 语法分类：句型 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**There be** 表示“**某处存在某人/某物**”，**不表示所属关系**。句首的 **There** 只是引导词，**没有“那里”的意思**；真正的主语是 **be** 后面的名词（主语）。

- **There is a park near my home.** 我家附近有一个公园。（强调“存在”）
- 若要说“某人拥有某物”，常用 **have / has**（所属）：**My uncle has a small boat.** 我叔叔有一艘小船。

---

### 2. 构成形式（与不同时态搭配）

| 时态 | 结构要点 | 示例 |
|------|-----------|------|
| **一般现在时** | **There is + 单数/不可数**；**There are + 复数** | There **is** some juice. / There **are** three cups. |
| **一般过去时** | **There was / There were** | There **was** a party last night. / There **were** many people. |
| **一般将来时** | **There will be…** 或 **There is / are going to be…**（**going to** 前的 **is/are** 仍按**后面第一个主语**判断） | There **will be** a test tomorrow. / There **is** going to be a meeting. |
| **现在完成时** | **There has been + 单数/不可数**；**There have been + 复数** | There **has been** an accident. / There **have been** two concerts. |

**否定与疑问**：把 **be**（或 **will / have** 等助动词）提前或加 **not**，规律与普通句中的 **be** 一致。  
**一般疑问**（现在时）：**Is there…? / Are there…?**  
**现在完成疑问**：**Has there been…? / Have there been…?**

---

### 3. 用法与语境

**用法一：There be 表“存在”，have / has 表“拥有”**  
- **There is a hole in my pocket.** 我口袋上有个洞。（客观上“有”）  
- **I have a hole in my pocket.** 我口袋上有个洞。（侧重“我这边有/带着”）  
中考常考：**because there is / are…** 说明原因（现场存在什么）。

**用法二：就近原则（主谓一致）**  
**be** 的形式由**最靠近它的那个名词**决定：  
- **There is a pen and two books on the desk.**（靠近 **a pen**，用 **is**）  
- **There are two books and a pen on the desk.**（靠近 **two books**，用 **are**）

**用法三：There be + 现在分词（doing）——有人/物正在做**  
- **There is a boy swimming in the river.** 河里有个男孩在游泳。（**a boy** 与 **swimming** 是逻辑上的主谓关系）  
- **There are some children playing outside.** 外面有一些孩子在玩。

**用法四：There be + 不定式（to do）——有事要做/可说**  
- **There is nothing to worry about.** 没什么好担心的。（**to worry about** 修饰 **nothing**）  
- **There is a lot of homework to finish tonight.** 今晚有很多作业要完成。

**用法五：There will be / There is going to be 表将来安排**  
- **There will be a sports meeting next week.**  
- **There is going to be a parents’ meeting on Friday.**

---

### 4. 标志词 / 关键信号

| 标志词 / 语境 | 说明 | 示例 |
|--------------|------|------|
| **now / today / every day** | 常用 **There is / are** | There **are** few students in the classroom now. |
| **yesterday / last… / …ago** | 用 **There was / were** | There **was** no bus ten minutes ago. |
| **tomorrow / next… / soon** | 用 **will be / is going to be** | There **will be** a show next Sunday. |
| **since / for / already / yet**（与完成时呼应） | 用 **There has / have been** | There **have been** many changes since then. |
| **Look! / Listen!** | 常接 **There is / are… doing** | Look! There **is** a cat climbing the tree. |

---

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *There **have** a library.* | **There is** a library. | 表“存在”用 **There be**，不用 **have**。 |
| 2 | *There **is** two boys.* | There **are** two boys. | 主语为复数，**be** 用 **are**。 |
| 3 | *There **is** going to **have** a party.* | There **is** going to **be** a party. | “将有”是 **there be** 的将来式，不能写成 **there have**。 |
| 4 | *There is nothing **worrying** about.*（想表达“无需担心”） | There is nothing **to worry about**. | 表“没什么可……”常用 **nothing / something + to do**。 |
| 5 | *Because **it has** no chairs…*（强调房间里“没有椅子”） | Because **there are** no chairs… | 描述空间内“有/没有”某物，多用 **there be**。 |

### 逐题解析

**第 1 题：答案 A**

- **解析**：**There be** 与多个主语连用时，**be** 的形式常按**最靠近它的名词**决定；本句紧贴 **be** 的是 **a pen**（单数），故用 **There is**。
- **排除法**：B **are** 与 **a pen** 不一致；C、D 为过去时，与语境“书桌上现在的状态”不符。
- **考查知识点**：知识点精讲 → **2. 构成形式（一般现在时）**、**3. 用法二：就近原则**。

**第 2 题：答案 B**

- **解析**：时间标志 **yesterday afternoon** 表示过去，主语 **many students** 为复数，故用 **There were**。
- **排除法**：A 为现在时；C 为现在完成时，与单纯过去时间点不搭配；D 为将来时。
- **考查知识点**：知识点精讲 → **2. 一般过去时：There was / were**、**4. 标志词（yesterday）**。

**第 3 题：答案 C**

- **解析**：**next month** 表将来，表示“将有”用 **There will be** 或 **There is going to be**；选项中只有 **will be** 符合。
- **排除法**：A、B 为现在/过去，与将来不符；D **has been** 表“从过去持续到现在”，与 **next month** 矛盾。
- **考查知识点**：知识点精讲 → **2. 一般将来时**、**3. 用法五**。

**第 4 题：答案 D**

- **解析**：询问“是否存在”，且主语 **milk** 为不可数名词，现在时的一般疑问句用 **Is there…?**
- **排除法**：**Have/Has** 用于 **There have/has been**，不接单纯 **any milk** 的现在存在问法；**Does there** 不是正确结构。
- **考查知识点**：知识点精讲 → **2. 一般疑问句（Is there…?）**。

**第 5 题：答案 A**

- **解析**：**no letters** 为复数概念，**this morning** 表过去，故用 **There were**。
- **排除法**：B **is** 与复数 **letters** 不一致；C **have** 不能用于 **There have a…** 表存在；D 将来时与 **this morning** 不符。
- **考查知识点**：知识点精讲 → **5. 易错点 1**（存在用 **be**）、**2. 一般过去时**。

**第 6 题：答案 B**

- **解析**：第一空表示“姐姐拥有一只狗”，用 **has**；第二空表示“花园里存在一棵树”，用 **there is**。
- **排除法**：A 把 **is** 与 **has** 颠倒；C 语序混乱；D 第二空 **has** 不能表“某地有某物”的 **there be** 结构。
- **考查知识点**：知识点精讲 → **3. 用法一：There be 与 have/has 的区别**。

**第 7 题：答案 C**

- **解析**：**since 2020** 与现在完成时呼应；主语 **many changes** 为复数，用 **There have been**。
- **排除法**：A、B 不是完成时结构；D **was** 无法与 **been** 构成完成时。
- **考查知识点**：知识点精讲 → **2. 现在完成时：There have been + 复数**、**4. 标志词（since）**。

**第 8 题：答案 D**

- **解析**：**Look!** 提示正在发生；**a boy** 为单数，用 **There is**；“男孩正在游泳”用 **swimming** 作后置说明。
- **排除法**：A 不能用原形 **swim** 直接接在名词后表进行；B **are** 与 **a boy** 不一致；C **swims** 不作后置进行用法。
- **考查知识点**：知识点精讲 → **3. 用法三：There be + doing**。

**第 9 题：答案 A**

- **解析**：**nothing to worry about** 为固定表达，“没有什么可担心的”，用不定式 **to worry about** 修饰 **nothing**。
- **排除法**：B、C、D 不符合该结构的地道用法。
- **考查知识点**：知识点精讲 → **3. 用法四：There be + to do**、**5. 易错点 4**。

**第 10 题：答案 B**

- **解析**：**there be** 与 **A and B** 连用时，**be** 常按**紧靠它的第一个名词**变化；**some water** 为不可数，故用 **There is**。
- **排除法**：A **are** 与不可数 **water** 不一致；C 过去时无时间依据；D **have been** 与语境“描述袋内现状”不符。
- **考查知识点**：知识点精讲 → **3. 用法二：就近原则**。

**第 11 题：答案 C**

- **解析**：**When we got to…** 表示过去某一时刻，**tickets** 为复数，故 **There were**。
- **排除法**：A、B 为现在时；D 将来时与“到达时”矛盾。
- **考查知识点**：知识点精讲 → **2. 一般过去时**、**4. 标志词（过去语境）**。

**第 12 题：答案 D**

- **解析**：**There is going to be…** 中 **is** 与 **going to be** 搭配；后面 **a parents’ meeting** 为单数，用 **is**（**are going to be** 若最近主语为复数也可用，本题选项设计为 **is**）。
- **排除法**：A **will** 后不能直接接 **going to**；B **are** 与 **a parents’ meeting** 单数不一致；C **have** 结构错误。
- **考查知识点**：知识点精讲 → **2. 一般将来时：There is going to be**、**5. 易错点 3**。

**第 13 题：答案 A**

- **解析**：**…been any phone calls** 为现在完成时疑问；**calls** 复数，用 **Have there been…?**
- **排除法**：B **Has** 与复数 **calls** 不一致；C、D **Were/Was there been** 为错误结构。
- **考查知识点**：知识点精讲 → **2. 现在完成时疑问：Have there been…**。

**第 14 题：答案 B**

- **解析**：说明房间内部“没有窗户”这种客观存在，用 **there are no windows**；**windows** 为复数。
- **排除法**：A **it has no windows** 在部分语境可接受，但初中阶段强调“空间内有/无”时常考 **there be**；C **is** 与复数 **windows** 不一致；D 语义不通。
- **考查知识点**：知识点精讲 → **3. 用法一**、**5. 易错点 5**。

**第 15 题：答案 C**

- **解析**：**In ten years** 表将来，用 **There will be**。
- **排除法**：A、B 现在时；D 过去时。
- **考查知识点**：知识点精讲 → **2. 一般将来时**、**4. 标志词（In ten years）**。

**第 16 题：答案 D**

- **解析**：**now** 与 **only grass**（不可数）搭配，用 **There is**。
- **排除法**：A **are**、B **were** 与不可数 **grass** 不一致；C **have been** 与“现在只有草”的简单现状相比，此处用一般现在时更自然。
- **考查知识点**：知识点精讲 → **2. 一般现在时：There is + 不可数**。

**第 17 题：答案 A**

- **解析**：靠近 **be** 的是 **two maps**（复数），故 **There are**。
- **排除法**：B **is**、C **was** 与复数主语不一致；D 完成时无相应时间标志。
- **考查知识点**：知识点精讲 → **3. 用法二：就近原则**。

**第 18 题：答案 B**

- **解析**：条件句表示“如果没有足够的椅子（供大家坐）”，指现场是否存在足够数量，用 **there aren’t enough chairs**。
- **排除法**：A、D 主语与“椅子是否存在”不匹配；C **isn’t** 与复数 **chairs** 不一致。
- **考查知识点**：知识点精讲 → **3. 用法一（存在）**、**5. 易错点 5**。

**第 19 题：答案 C**

- **解析**：**Since last summer** 表示从过去某时持续到现在，用 **There has been**（**a new library** 单数）。
- **排除法**：A、B 不能体现 **since** 的完成时意义；D 将来时与 **since** 矛盾。
- **考查知识点**：知识点精讲 → **2. There has been + 单数**、**4. 标志词（since）**。

**第 20 题：答案 D**

- **解析**：说明公交车站“存在着一条长队”，用 **there was**；**a long line** 为单数，且迟到原因发生在过去。
- **排除法**：A **it had a long line** 不自然；B **I have** 侧重所属，不符合；C **there were** 与单数 **a long line** 不一致。
- **考查知识点**：知识点精讲 → **3. 用法一：表存在**、**2. 一般过去时**。
