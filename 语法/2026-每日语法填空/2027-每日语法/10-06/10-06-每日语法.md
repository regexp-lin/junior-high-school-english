# 主谓一致——不定代词与集合名词作主语（Subject-Verb Agreement — Indefinite Pronouns & Collective Nouns）

> 日期：2027 年 10 月 6 日 | 语法分类：句型 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

**主谓一致（subject–verb agreement）**指句子的**谓语动词在人称和数上要与主语保持一致**。当主语是**不定代词**或**集合名词**时，不能只看“中文感觉像很多人”，而要按英语习惯判断主语表示**单一概念**还是**多个成员**，从而选用**单数**或**复数**谓语。

---

### 2. 构成形式

| 主语类型 | 结构要点 | 谓语动词 | 示例 |
|---------|---------|---------|------|
| **不定代词（表单数概念）** | **everyone / everybody / someone / somebody / anyone / anybody / no one / nobody / each / either / neither** 等作主语 | 常用**单数** | **Everyone** **is** here. |
| **both** | **Both (of…)** 作主语，表示“两者都” | 用**复数** | **Both** **are** ready. |
| **集合名词（整体）** | **family / class / team / group** 等被看作**一个单位** | 常用**单数** | Our class **is** big. |
| **集合名词（成员）** | 强调**里面的成员各自行动** | 常用**复数** | The class **are** wearing different uniforms today. |
| **police** | 形式单数，常作**复数意义的集体** | 谓语多用**复数** | The police **are** helping. |
| **the number of + 复数名词** | 中心词是 **number**（……的数量） | **单数**谓语 | The number of books **is** 50. |
| **a number of + 复数名词** | 相当于 **many**（许多……） | **复数**谓语 | A number of books **are** on sale. |
| **each of / one of + 复数名词** | 强调**其中之一 / 每一个** | **单数**谓语 | Each of the boys **has** a map. |

---

### 3. 用法与语境

**用法一：表单数概念的不定代词作主语 → 谓语用单数**  
- **Everyone** in our grade **knows** the safety rules.  
  我们年级的每个人都知道安全守则。  
- **Nobody** **was** at home when I called.  
  我打电话时没有人在家。

**用法二：both 作主语 → 谓语用复数**  
- **Both** of my friends **like** science fiction.  
  我的两个朋友都喜欢科幻作品。

**用法三：集合名词——看“整体”还是“成员”**  
- My **family** **is** moving to a new flat next month.（把家庭当作一个整体）  
  我家下个月要搬到新公寓。  
- The **team** **are** arguing about the plan.（强调队员们在争论）  
  队员们在争论这个计划。

**用法四：the number of 与 a number of**  
- **The number of** volunteers **grows** every year.（数量在增长，主语核心是 number）  
  志愿者的数量每年都在增长。  
- **A number of** volunteers **help** at the weekend.（许多志愿者，主语相当于 volunteers）  
  许多志愿者在周末帮忙。

**用法五：each of / one of + 复数名词 → 谓语仍用单数**  
- **One of** my classmates **plays** the guitar very well.  
  我的一位同班同学吉他弹得很好。  
- **Each of** the students **has** a locker.  
  每个学生都有一个储物柜。

---

### 4. 标志词 / 关键信号

| 标志词 / 结构 | 说明 | 示例 |
|--------------|------|------|
| **every- / some- / any- / no- + one/body/thing** | 常表单数概念 | **Something** **is** wrong. |
| **each / either / neither** | 常接单数谓语 | **Neither** answer **is** OK. |
| **both … and …** | 连接两个主语时谓语用复数 | Both Tom and Jim **are** here. |
| **the number of** | 后接复数名词，谓语仍**单数** | The number of girls **is** 12. |
| **a number of** | 后接复数名词，谓语**复数** | A number of girls **are** dancing. |
| **each of / one of + 复数** | 谓语**单数** | One of the keys **fits** the door. |
| **family / class / team / group + 整体描述** | 常用 **is/has** 等单数 | The team **has** a new coach. |

---

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *Everyone **are** happy.* | Everyone **is** happy. | **Everyone** 表单数概念，谓语用单数。 |
| 2 | *Both **is** correct.* | Both **are** correct. | **Both** 指两个都，谓语用复数。 |
| 3 | *A number of students **is** late.* | A number of students **are** late. | **a number of** 表“许多”，主语概念上是复数。 |
| 4 | *The number of students **are** 40.* | The number of students **is** 40. | 主语是 **the number**，不是 students。 |
| 5 | *Each of the boys **have** a ball.* | Each of the boys **has** a ball. | **Each of + 复数名词** 仍表单数意义。 |

### 逐题解析

**第 1 题：答案 A**

- **解析**：主语是 **Everyone**（每个人），属于表单数概念的不定代词，谓语动词用第三人称单数 **is**。
- **排除法**：**are** 与单数主语不一致；**am** 只与 **I** 搭配；**be** 为原形，不能单独作谓语。
- **考查知识点**：知识点精讲 → 用法一；易错点 1。

**第 2 题：答案 B**

- **解析**：**Both of the twins** 表示“两个都”，主语为复数概念，谓语用 **are**。
- **排除法**：**is / am / be** 均不能与 **both** 作主语时的复数意义一致。
- **考查知识点**：知识点精讲 → 用法二；构成形式表中 **both** 一行。

**第 3 题：答案 C**

- **解析**：**The number of books** 的中心词是 **number**（……的数量），谓语用单数 **is**。
- **排除法**：**are / were** 误把 **books** 当成了主语核心；**have** 与 “数量在变大” 的搭配不当。
- **考查知识点**：知识点精讲 → 用法四；易错点 4。

**第 4 题：答案 D**

- **解析**：**A number of parents** 表示“许多家长”，相当于复数主语，谓语用 **are**。
- **排除法**：**is / was** 为单数；**has** 后不能接 **waiting** 表“正在等候”。
- **考查知识点**：知识点精讲 → 用法四；易错点 3。

**第 5 题：答案 A**

- **解析**：句意强调“家庭”作为一个整体单位，**My family** 作单数概念，用 **is**。
- **排除法**：若强调家庭成员各自不同，才可能用 **are**；本题“a small one”指整体。
- **考查知识点**：知识点精讲 → 用法三（集合名词整体）。

**第 6 题：答案 B**

- **解析**：**The class** 后接 **discussing … in groups**，强调班里同学分小组讨论，侧重**成员活动**，谓语常用复数 **are**。
- **排除法**：**is** 更偏“班级整体被描述”；**was / has been** 与 **now** 语境不符。
- **考查知识点**：知识点精讲 → 用法三（成员）。

**第 7 题：答案 C**

- **解析**：**Neither of the boys** 中 **neither** 表单数“两者都不”，谓语用单数 **has**（在学校上课期间没有手机）。
- **排除法**：**have / were** 与 **neither of** 的单数一致规则不符；**are** 语义也不搭配 **a phone**。
- **考查知识点**：构成形式表；用法一（与 **nobody** 同类的不定代词思维）。

**第 8 题：答案 D**

- **解析**：**Each of the volunteers** 表单数“每一位”，谓语用第三人称单数；**wears** 与 **a name tag** 搭配自然。
- **排除法**：**have / wear** 为复数形式；**has** 与“佩戴胸牌”不如 **wears** 贴切。
- **考查知识点**：知识点精讲 → 用法五；易错点 5。

**第 9 题：答案 A**

- **解析**：**Our school basketball team** 在此表示**队伍整体**的战绩，视作单数单位，用 **has won**。
- **排除法**：**have** 用于强调整体中的成员时更常见，本题“球队本赛季赢了多场”偏整体成绩；**are / were** 与 **won** 结构不搭。
- **考查知识点**：知识点精讲 → 用法三（整体）。

**第 10 题：答案 B**

- **解析**：**Someone** 作主语，谓语用单数 **has left**（现在完成时表对现在的影响：桌上多了本子）。
- **排除法**：**have / are / were** 与 **someone** 的单数规则不符。
- **考查知识点**：用法一；标志词表中的 **some-** 一类。

**第 11 题：答案 C**

- **解析**：**One of my cousins** 后接单数谓语，且 **now** 表现阶段状态，用一般现在时 **lives**。
- **排除法**：**live** 缺第三人称 **-s**；**are living** 把主语误作复数；**has lived** 偏重经历，与 **now** 略不自然。
- **考查知识点**：用法五 **one of + 复数名词**。

**第 12 题：答案 D**

- **解析**：**Either you or Lily** 结构中，谓语常按**邻近原则**与靠近动词的主语一致，**Lily** 为第三人称单数，用 **is**。
- **排除法**：**am** 只搭配 **I**；**are** 与 **Lily** 不一致；**be** 不能单独作谓语。
- **考查知识点**：拔高题：并列连词 **either … or …** 与主谓一致（初中常考邻近一致）。

**第 13 题：答案 A**

- **解析**：**The group was divided into two teams** 把 **group** 看作**一个整体**接受安排，被动结构中单数更自然，用 **was**。
- **排除法**：**were** 有时用于强调成员，但本题“被分成两队”强调整体被划分；**are / have been** 与语境时态不符。
- **考查知识点**：集合名词整体与被动的综合运用（拔高）。

**第 14 题：答案 B**

- **解析**：**Everybody** 表单数概念，谓语用单数；**when we started** 表过去，用 **was**。
- **排除法**：**were / are / have been** 与 **everybody** 的单数要求不一致。
- **考查知识点**：用法一（**everyone/everybody**）。

**第 15 题：答案 C**

- **解析**：**Nobody** 作主语用单数；**before the teacher announced** 表示“在宣布之前”，**known** 发生在 **announced** 之前，用过去完成时 **had known**。
- **排除法**：**have / has** 时态与“先于 announced”不符；**have had** 结构冗余。
- **考查知识点**：**nobody** 的单数一致 + 时态（拔高）。

**第 16 题：答案 D**

- **解析**：句意要表达“我们队里**许多**队员来自同一所初中”，用 **A number of … are**。**The number of** 表“……的数量”，与句意不符。
- **排除法**：**The number … is** 表数量；**The number … are** / **A number … is** 均违背固定搭配的主谓一致规则。
- **考查知识点**：用法四；易错点 3 与 4 的对比。

**第 17 题：答案 A**

- **解析**：**Each student** 表单数，谓语用 **has**（有机会）。
- **排除法**：**have / are / were** 与 **each** 的单数意义不符。
- **考查知识点**：**each** 作主语；用法五。

**第 18 题：答案 B**

- **解析**：**The police** 在英语中常视为**复数集体**，谓语用 **are**。
- **排除法**：**is / was / has been** 不符合 **police** 的常见用法。
- **考查知识点**：构成形式表中 **police** 一行。

**第 19 题：答案 C**

- **解析**：**Not only … but also …** 连接主语时，谓语常遵循**邻近一致**，靠近谓语的是 **my aunt**，用 **is**。
- **排除法**：**are / were / have been** 与 **aunt** 不一致。
- **考查知识点**：拔高题：并列主语与邻近一致。

**第 20 题：答案 D**

- **解析**：**Neither** 表示“两者都不”，表单数概念，**Neither answer** 后用 **is**。
- **排除法**：**are / were / have been** 均为复数形式，与 **neither** 不符。
- **考查知识点**：构成形式表 **neither**；易错点 1 同类思路。
