# 定语从句——关系代词 that 与 which 的区别（Attributive Clauses — that vs. which）

> 日期：2027 年 11 月 20 日 | 语法分类：从句 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

定语从句（Attributive Clause / Relative Clause）是修饰名词或代词的从句，放在被修饰词（先行词）之后。引导定语从句的关系代词有 that、which、who、whom、whose 等。其中 **that** 和 **which** 都可以指代"物"，但在某些情况下**只能用 that**或**只能用 which**，这是中考的重要考点。

### 2. 构成形式

| 关系代词 | 指代 | 在从句中的成分 | 示例 |
|---------|------|-------------|------|
| that | 人或物 | 主语/宾语 | The boy that is reading is Tom. |
| which | 物 | 主语/宾语 | The book which I bought is interesting. |
| who | 人 | 主语 | The girl who is singing is my sister. |
| whom | 人 | 宾语 | The man whom you met is my uncle. |
| whose | 人或物（的） | 定语 | The boy whose father is a doctor is Tom. |

| 句型 | 结构 | 示例 |
|------|------|------|
| 关系代词作主语 | 先行词 + that/which + 谓语 + ... | The dog that is running is mine. |
| 关系代词作宾语 | 先行词 + (that/which) + 主语 + 谓语 + ... | The movie (that) I saw was great. |

### 3. 用法与语境

- **用法一**：that 和 which 都可以指代物，在一般情况下可互换
  - 例：The book that/which is on the desk is mine.（桌上的那本书是我的。）
  - 例：I like the cake that/which my mom made.（我喜欢我妈做的蛋糕。）

- **用法二**：只能用 that 不能用 which 的情况
  - ① 先行词是不定代词（all, everything, nothing, something, anything, much, little, few 等）
  - ② 先行词被最高级修饰
  - ③ 先行词被序数词修饰
  - ④ 先行词被 the only, the very, the same, the last 等修饰
  - ⑤ 先行词既有人又有物
  - 例：Everything that he said was true.（他说的一切都是真的。）
  - 例：This is the best movie that I've ever seen.（这是我看过的最好的电影。）

- **用法三**：只能用 which 不能用 that 的情况
  - ① 在非限制性定语从句中（有逗号隔开）
  - ② 介词后面
  - 例：His new car, which he bought last month, is very nice.（他上月买的新车很漂亮。）
  - 例：The house in which he lives is very old.（他住的房子很旧。）

- **用法四**：关系代词作宾语时可以省略
  - 例：The movie (that/which) I watched last night was exciting.（我昨晚看的电影很刺激。）

### 4. 标志词 / 关键信号

| 标志词/短语 | 说明 | 示例 |
|------------|------|------|
| all / everything / nothing | 只用 that | Is there anything that I can do? |
| the best / the most... | 最高级修饰只用 that | the longest river that I know |
| the first / the second | 序数词修饰只用 that | the first thing that you should do |
| the only / the very | 只用 that | the only book that I have |
| 人 + 物（并列） | 只用 that | the man and his dog that... |
| 逗号 + 从句 | 只用 which | ..., which is very nice. |
| 介词 + 关系代词 | 只用 which | in which, from which |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-----------|-----------|---------|
| 1 | Everything which he said was wrong. | Everything that he said was wrong. | 先行词是 everything，只能用 that |
| 2 | This is the best film which I have seen. | This is the best film that I have seen. | 先行词有最高级修饰，只能用 that |
| 3 | He has a car, that is very expensive. | He has a car, which is very expensive. | 非限制性定语从句（有逗号）只能用 which |
| 4 | The pen with that he writes is new. | The pen with which he writes is new. | 介词后只能用 which，不能用 that |
| 5 | Is there something that you want? | Is there anything that you want? | 疑问句/否定句中用 anything 而非 something |

### 逐题解析

**第 1 题：答案 B**

- **解析**：先行词 the book 是物，关系代词在从句中作主语，用 which 或 that。选 B。
- **排除法**：A. who 指人；C. whom 作宾语；D. whose 表所有关系。
- **考查知识点**：知识点精讲 → 用法一：that/which 指代物。

**第 2 题：答案 C**

- **解析**：先行词是 Everything（不定代词），只能用 that。
- **排除法**：A. which 不能用于不定代词后；B. who 指人；D. whom 指人。
- **考查知识点**：知识点精讲 → 用法二①：不定代词只用 that。

**第 3 题：答案 C**

- **解析**：先行词有最高级 most interesting 修饰，只能用 that。
- **排除法**：A. which 不能用于最高级后；B/D 指人。
- **考查知识点**：知识点精讲 → 用法二②：最高级修饰只用 that。

**第 4 题：答案 B**

- **解析**：先行词 the girl 是人，关系代词作主语，用 who。
- **排除法**：A. which 指物；C. whom 作宾语；D. whose 表所有。
- **考查知识点**：知识点精讲 → 构成形式：who 指代人作主语。

**第 5 题：答案 D**

- **解析**：先行词是 anything（不定代词），只能用 that。
- **排除法**：A. which 不用于不定代词后；B. what 不引导定语从句；C. who 指人。
- **考查知识点**：知识点精讲 → 用法二①：不定代词只用 that。

**第 6 题：答案 C**

- **解析**：先行词 the pen 是物，关系代词在从句中作 bought 的宾语，用 that（也可省略）。
- **排除法**：A/B 指人；D. whose 表所有。
- **考查知识点**：知识点精讲 → 用法四：关系代词作宾语。

**第 7 题：答案 C**

- **解析**：先行词被 the only 修饰，只能用 that。虽然先行词是人，但 the only 限定下优先 that。
- **排除法**：A. which 指物；B. who 可以但 the only 修饰下 that 更优；D. whom 作宾语。
- **考查知识点**：知识点精讲 → 用法二④：the only 修饰只用 that。

**第 8 题：答案 A**

- **解析**：先行词 the music 是物，关系代词作从句主语，that 或 which 都行，选 A。
- **排除法**：B/C/D 指人。
- **考查知识点**：知识点精讲 → 用法一。

**第 9 题：答案 C**

- **解析**：先行词被序数词 the first 修饰，只能用 that。
- **排除法**：A. which 不用于序数词后；B. when 不对；D. where 不对。
- **考查知识点**：知识点精讲 → 用法二③：序数词修饰只用 that。

**第 10 题：答案 B**

- **解析**：先行词 the man 是人，关系代词作从句主语，that 或 who 都可以，选 B。
- **排除法**：A. which 指物；C. whom 作宾语；D. whose 表所有。
- **考查知识点**：知识点精讲 → 构成形式：that 也可指代人。

**第 11 题：答案 C**

- **解析**：先行词既有人（the boy）又有物（his dog），只能用 that。
- **排除法**：A. which 只指物；B. who 只指人；D. whom 只指人。

**第 12 题：答案 B**

- **排除法**：A. that 不用于非限制性定语从句；C/D 指人。
- **考查知识点**：知识点精讲 → 用法三①：非限制性定语从句只用 which。

**第 13 题：答案 B**

- **解析**：介词 in 后面只能用 which，不能用 that。
- **排除法**：A. that 不用于介词后；C. who 指人；D. where 虽表地点但此处有介词 in。

- **考查知识点**：知识点精讲 → 拔高：the reason + why/that。

**第 16 题：答案 C**

- **排除法**：A/B 虽语法上可以但 when 更自然；D. where 表地点。
- **考查知识点**：知识点精讲 → 拔高：先行词是时间用 when。

**第 17 题：答案 C**

- **排除法**：A. which 不用于 the last 后；B/D 指人。
- **考查知识点**：知识点精讲 → 用法二④：the last 修饰只用 that。

**第 18 题：答案 B**

- **解析**：Tom's mother → whose mother，whose 表所有关系。注意有逗号是非限制性定语从句。
- **排除法**：A. who 不表所有；C. that 不用于非限制性；D. which 指物。
- **考查知识点**：知识点精讲 → 构成形式：whose 表所有关系。

**第 19 题：答案 C**
