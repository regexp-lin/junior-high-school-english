# so / neither 倒装——So do I / Neither do I（Inversion with so / neither）

> 日期：2027 年 02 月 21 日 | 语法分类：句法 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

当上一句表述了某种情况后，如果另一个人"也如此"或"也不如此"，英语用 **so / neither (nor)** 引导的**倒装句**来表达，避免重复。

> —I like music. —**So do I.**（我也喜欢。）
> —I can't swim. —**Neither can I.**（我也不会。）

---

### 2. 构成形式

#### 2.1 "也"——So + 助动词/be/情态动词 + 主语

用于前句是**肯定句**，后句表示"……也是如此"。

| 前句类型 | So 倒装 | 示例 |
|---------|---------|------|
| be 动词 | So + be + 主语 | —He is happy. —**So am I.** |
| 实义动词（一般现在） | So + do/does + 主语 | —She likes tea. —**So do I.** |
| 实义动词（一般过去） | So + did + 主语 | —They went there. —**So did we.** |
| 情态动词 | So + 情态动词 + 主语 | —He can swim. —**So can she.** |
| 现在完成时 | So + have/has + 主语 | —I have been to Beijing. —**So has Tom.** |
| will / shall | So + will/shall + 主语 | —I will go. —**So will I.** |

#### 2.2 "也不"——Neither (Nor) + 助动词/be/情态动词 + 主语

用于前句是**否定句**，后句表示"……也不如此"。

| 前句类型 | Neither 倒装 | 示例 |
|---------|-------------|------|
| be 动词 | Neither + be + 主语 | —I'm not tired. —**Neither am I.** |
| 实义动词（一般现在） | Neither + do/does + 主语 | —He doesn't like fish. —**Neither do I.** |
| 实义动词（一般过去） | Neither + did + 主语 | —She didn't go. —**Neither did I.** |
| 情态动词 | Neither + 情态动词 + 主语 | —Tom can't drive. —**Neither can I.** |
| 现在完成时 | Neither + have/has + 主语 | —I haven't seen the film. —**Neither has she.** |

> **nor** 可以替代 neither，用法完全相同：**Nor do I.**

---

### 3. 特殊情况：So + 主语 + 助动词（不倒装）

如果主语**与前句是同一人**，表示"确实如此"，不倒装。

| 结构 | 含义 | 示例 |
|------|------|------|
| So + 主语 + 助动词 | 确实如此 | —He is clever. —**So he is.**（他确实聪明。）|

**对比：**
- —He is clever. —**So am I.**（我也聪明。← 不同人，倒装）
- —He is clever. —**So he is.**（他确实聪明。← 同一人，不倒装）

---

### 4. 关键规则

1. **助动词/be/情态动词的选择**必须与前句一致。
2. **时态**必须与前句一致。
3. **主语**必须是另一个人或事物（倒装句）；同一个人时不倒装。
4. 前句肯定用 **so**，前句否定用 **neither / nor**。

---

### 5. 标志词 / 关键信号

| 标志 | 说明 |
|------|------|
| So + 助动词 + 主语 | 某人也如此（前句肯定） |
| Neither / Nor + 助动词 + 主语 | 某人也不如此（前句否定） |
| So + 主语 + 助动词 | 确实如此（同一人，不倒装） |
| Me too. / Me neither. | 口语简化形式 |

---

### 6. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|-------------|-------------|----------|
| 1 | —I like music. —So like I. | —**So do I.** | 倒装句用助动词 do，不用实义动词 |
| 2 | —He is a student. —So do I. | —**So am I.** | 前句用 be 动词，后句也用 be |
| 3 | —He can swim. —So I can. (想说"我也会") | —**So can I.** | 不同人用倒装 |
| 4 | —She didn't go. —So didn't I. | —**Neither did I.** | 前句否定用 neither，不用 so |
| 5 | —He has finished. —Neither have I. | —**So have I.** | 前句肯定用 so，不用 neither |

### 逐题解析

**第 1 题：答案 A**

- **解析**：前句 I like（肯定，一般现在时实义动词），后句用 **So do I**。
- **排除法**：B 用 be 不对；C neither 用于否定；D 不倒装表"确实如此"。
- **考查知识点**：So + do + 主语。

**第 2 题：答案 B**

- **解析**：前句 He is（be 动词），后句用 **So am I**。
- **排除法**：A 用 do 不对；C 用 neither 不合；D 不倒装。
- **考查知识点**：So + be + 主语。

**第 3 题：答案 B**

- **考查知识点**：Neither + 情态动词 + 主语。
