# it 的用法——形式主语和形式宾语（Uses of "It" — Formal Subject & Formal Object）

> 日期：2027 年 8 月 28 日 | 语法分类：特殊结构 | 难度：中等 + 拔高

---

## 一、知识点精讲

### 1. 基本概念

英语里 **it** 不只是“它”。当中考题干出现 **to do / that 从句** 很长、却要放在**句首或动词后**时，常用 **it** 来“占位”，真正的意思由后面的 **不定式或从句**承担：

- **形式主语（formal subject）**：**It** 放在句首，看起来像主语，但**真正的主语**在后面（多为 **to do** 或 **that 从句**）。
- **形式宾语（formal object）**：在 **think / find / make / feel** 等动词后，**it** 作宾语“占位”，**真正的宾语**是后面的 **to do** 或 **that 从句**；若还有宾语补足语，结构常是 **动词 + it + 形容词/名词 + to do / that…**。

另外，**it** 还可指**天气、时间、距离**等，这时 **it** 有**实际指代意义**，不是“形式”用法。

### 2. 构成形式

| 类型 | 结构 | 示例 |
|------|------|------|
| **形式主语：It is + adj. + to do** | **It is / was + 形容词 + to do** | *It is important **to review** notes.* |
| **形式主语：It is + adj. + that…** | **It is / was + 形容词 + that 从句** | *It is clear **that he tried his best**.* |
| **形式主语：It + 系动词 + 名词 + to do / that…** | **It is / was + a(n) + 名词 + to do / that…** | *It is a pity **to miss** the show.* |
| **It takes (sb.) + 时间 + to do** | **It takes / took + (人) + 时间 + to do** | *It took me an hour **to finish**.* |
| **评价/传闻类形式主语** | **It seems / appears / sounds + that…**；**It is said / reported + that…** | *It seems **that she is right**.* |
| **形式宾语：find / think + it + adj. + to do** | **动词 + it + 形容词 + to do** | *I find it hard **to get up** early.* |
| **形式宾语：make + it + clear / a rule + that…** | **make + it + 补语 + that 从句** | *He made it clear **that he disagreed**.* |
| **天气 / 时间 / 距离** | **It** + 谓语（表天气、钟点、路程等） | *It is sunny.* / *It is 8 o’clock.* / *It is five miles.* |

### 3. 用法与语境

- **用法一：It is + 形容词 + to do（不定式作真正主语）**  
  英语习惯避免“头重脚轻”，所以把长的 **to do** 放后面，前面用 **It** 开头。  
  - 例：*It is necessary **to drink** more water in summer.*（夏天多喝水很有必要。）  
  - 例：*It was brave **of** him **to save** the little boy.*（他救那个小男孩很勇敢。）注意：**of sb.** 常表人的品质，**for sb.** 常表“对某人来说”。

- **用法二：It is + 形容词 + that 从句（that 从句作真正主语）**  
  **It** 仅占位，**that** 引导的从句才是主语。  
  - 例：*It is true **that practice makes perfect**.*（熟能生巧，这是真的。）  
  - 例：*It is surprising **that she finished** so much work in one day.*（她一天做完这么多活，真让人惊讶。）

- **用法三：It takes (sb.) some time to do**  
  表示“做某事花费（某人）多长时间”，**it** 为形式主语，**to do** 为真正主语。  
  - 例：*It usually takes me twenty minutes **to walk** to school.*（我步行到校一般要二十分钟。）  
  - 疑问形式：*Does it take you long **to read** this story?*

- **用法四：形式宾语——find / think / make / feel + it + 形容词/名词 + to do / that…**  
  当 **to do** 或 **that** 从句作宾语且后面还有**宾语补足语**时，常用 **it** 作形式宾语，把长结构后置。  
  - 例：*We think **it** helpful **to read** English every morning.*（我们认为每天早上读英语有帮助。）  
  - 例：*The coach made **it** a rule **that** everyone should warm up first.*（教练规定大家必须先热身。）  
  - 例：*I feel **it** my duty **to help** new classmates.*（我觉得帮助新同学是我的责任。）

- **用法五：it 指天气、时间、距离等**  
  此时 **it** 有具体含义，不是纯占位。  
  - 例：*It was raining hard when the class ended.*（下课时雨下得很大。）  
  - 例：*It is half past seven. Let’s hurry.*（七点半了，快点。）  
  - 例：*Is it far from your home to the library?*（你家到图书馆远吗？）

### 4. 标志词 / 关键信号

| 标志词 / 结构 | 说明 | 示例 |
|--------------|------|------|
| **It is / was + adj. + ______** | 空后多为 **to do** 或 **that** | *It is easy **to understand**.* |
| **It takes / took…** | 后接时间，再接 **to do** | *It took us two hours **to get** there.* |
| **find / think / make / feel + ______ + adj.** | 中间常为 **it** | *She finds **it** boring **to wait**.* |
| **It seems / sounds + that…** | **it** 作形式主语 | *It sounds **that you are** tired.*（也可用 *It sounds **as if**…*，初中以 **that** 类结构识记为主。） |
| **谈论天气、钟点、路程** | 句首 **It** | *It is cold today.* |
| **make it clear / make it a rule** | **it** 为形式宾语 | *Please make **it** clear **that**…* |

### 5. 易错点与辨析

| 序号 | ❌ 错误写法 | ✅ 正确写法 | 错因分析 |
|------|------------|------------|---------|
| 1 | *To learn English well is important for us.*（口语偶见，中考书面题常考规范句） | *It is important **for us to learn** English well.* | 用 **It** 作形式主语更自然、更符合初中考点。 |
| 2 | *I think **that** important to read every day.* | *I think **it** important **to read** every day.* | **think** 后需 **it** 作形式宾语，再接形容词与 **to do**。 |
| 3 | *He made **that** clear we should be quiet.* | *He made **it** clear **that** we should be quiet.* | **make it clear that…** 为固定型式，**it** 不可换成 **that**。 |
| 4 | *Is **that** far from here to the station?* | *Is **it** far from here to the station?* | 表距离用 **it** 作主语。 |
| 5 | *It took me two hours **finish** the task.* | *It took me two hours **to finish** the task.* | **It takes… to do** 中必须用 **不定式**。 |

### 逐题解析

**第 1 题：答案 A**

- **解析**：本句考查 **It is + adj. + to do**。**It** 作**形式主语**，真正的主语是后面的 **to learn English…**。
- **排除法**：**This / That / One** 不能用于该固定句型开头。
- **考查知识点**：知识点精讲 → **用法一**；**构成形式**表中“形式主语：It is + adj. + to do”。

**第 2 题：答案 B**

- **解析**：**It is said that…** 表示“据说……”，**It** 为形式主语，**that** 从句为真正主语。
- **排除法**：**This / That / What** 不能构成该固定结构。
- **考查知识点**：知识点精讲 → **构成形式**表中“评价/传闻类形式主语”及 **标志词**中的 **It is said…**。

**第 3 题：答案 C**

- **解析**：**It takes / took sb. + 时间 + to do** 中，真正主语为不定式，必须用 **to finish**。
- **排除法**：**finish / finishing / finished** 均不符合该句型。
- **考查知识点**：知识点精讲 → **用法三**；**易错点 5**。

**第 4 题：答案 D**

- **解析**：**find it + adj. + to do** 中 **it** 为形式宾语，**to plan…** 为真正宾语。
- **排除法**：**that / this / one** 不能作该结构中的形式宾语。
- **考查知识点**：知识点精讲 → **用法四**。

**第 5 题：答案 A**

- **解析**：**make it clear that…** 中 **it** 为形式宾语，**that** 从句为真正宾语。
- **排除法**：**that / this / one** 不能替换 **make it clear** 中的 **it**（参见易错点 3）。
- **考查知识点**：知识点精讲 → **用法四**；**易错点 3**。

**第 6 题：答案 B**

- **解析**：描述天气用 **It** 作主语：**It was raining**。
- **排除法**：**This / That / One** 不用于此语境。
- **考查知识点**：知识点精讲 → **用法五**。

**第 7 题：答案 C**

- **解析**：询问距离：**Is it far from A to B?**
- **排除法**：**this / that / one** 不用于该固定问法（易错点 4）。
- **考查知识点**：知识点精讲 → **用法五**；**易错点 4**。

**第 8 题：答案 D**
