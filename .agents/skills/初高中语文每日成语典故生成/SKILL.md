# 初高中语文每日成语典故生成技能

## 1. 核心目标

根据初高中语文学习要求，每天为学生生成 5 个常见的成语，并提供其释义与背后的历史典故，按日期保存到指定目录中。

## 2. 内容生成规范

- **成语选择**：必须是初高中阶段常见、易考且具有教育意义的成语。
- **内容要求**：
  - 提供准确的拼音。
  - 提供清晰易懂的释义。
  - 详细解释其背后的历史典故或来源（如出自哪部古籍、相关历史人物及故事），让学生既知其然又知其所以然。
- **去重机制**：每次生成前，必须扫描先前的目录（`junior-subjects/初中/语文/每日成语/` 及 `junior-subjects/初中/语文/2026-每日成语/` 下的所有日期文件夹及文件），提取已生成的成语列表，确保本次生成的 5 个成语不与之前生成的重复。

## 3. 文件生成与保存规则

1. **生成位置**：根据当前日期 `[MM-DD]`（例如：`03-18`），在路径 `/Users/teddy.lin/github/junior-subjects/初中/语文/每日成语/[MM-DD]/` 下创建或寻找对应目录。
2. **文件写入**：在该目录下生成 Markdown 文件写入这 5 个成语，文件命名如：`每日成语_[MM-DD].md`。
3. **内容格式**：
   - 文件顶部使用一级标题表明日期和主题，例如：`# 03-18 初高中每日成语`。
   - 正文依次列出 5 个成语，包含拼音、释义、典故。排版需清晰美观。
