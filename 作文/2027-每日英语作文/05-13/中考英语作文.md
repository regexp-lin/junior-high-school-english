# 中考英语作文生成

## 作文题目（情境+内容要求+注意事项）

假设你是李华，学校「传统文化周」英语征文活动请你围绕「中华传统美德」写一则短文。内容包括：

① 选择一种你认同的传统美德并简要说明其含义；  
② 结合生活或学习举一例说明该美德如何体现；  
③ 呼吁大家传承与践行这一美德。  

**注意：** 词数 80-100；文中不得出现真实姓名和校名。

---

### 题目信息

| 项目 | 内容 |
|------|------|
| 体裁 | 议论文（举例+呼吁） |
| 话题 | 中华传统美德（诚信） |
| 人称 | 第一人称 |
| 时态 | 一般现在时为主 |
| 字数 | 80-100词 |
| 要点数 | 3个 |

## 写作思考

### 审题
- **情境**：传统文化周英语征文。
- **身份**：李华。
- **核心任务**：选定美德 + 举例 + 呼吁传承。
- **限制**：80-100 词、无真实姓名校名。

### 结构安排
| 段落 | 内容 | 预计词数 |
|------|------|----------|
| 第一段 | 选择「诚信」：言行一致、守信 | 约30词 |
| 第二段 | 举例：捡到物品上交、考试不作弊 | 约35词 |
| 第三段 | 呼吁：从日常小事做起，传承美德 | 约30词 |

### 表达准备（三要点中英文）
| 要点 | 中文 | 英文表达参考 |
|------|------|--------------|
| ① | 诚实守信、说到做到 | honesty, keep one's word, tell the truth |
| ② | 拾金不昧、借东西按时归还 | return lost money, return things on time |
| ③ | 代代相传、做正直的人 | pass down the virtue, be an honest person |

### 可加分亮点（4条）
1. 用 *Among many virtues, I value ... most* 开篇。  
2. 用 *For example* 引出例证。  
3. 用 *In this way* 小结例子意义。  
4. 结尾 *Let's ...* 呼吁。

---

## 参考范文

Among many Chinese traditional virtues, I value honesty most. It means telling the truth, keeping promises and never taking what is not ours.

For example, when I found a wallet in the library, I handed it to the teacher right away. In exams, I never copy answers because fairness matters.

Honesty helps people trust one another and builds a better class. Let's pass down this virtue by doing small right things every day and encouraging friends to be upright.

---

## 范文亮点分析

### 内容方面
美德界定清楚；例证贴近学生（失物上交、考试诚信）；呼吁具体（日常小事、互相鼓励）。

### 结构方面
定义—例证—呼吁；第二段两个并列例子，信息充实而不杂乱。

### 语言方面
1. *keeping promises* — 信守承诺  
2. *never taking what is not ours* — 不拿不属于自己的东西  
3. *handed it to the teacher right away* — 立刻交给老师  
4. *fairness matters* — 公平很重要  
5. *trust one another* — 彼此信任  
6. *doing small right things* — 做正确的小事  

---

## 全文翻译

在众多中华传统美德中，我最看重诚信。它意味着讲真话、守承诺，不拿不属于自己的东西。

例如，当我在图书馆捡到一个钱包时，我立刻把它交给老师。在考试中，我从不抄袭答案，因为公平很重要。诚信能帮助人们彼此信任，建设更好的班级。让我们通过每天做正确的小事、鼓励朋友做正直的人，把这一美德传承下去。
