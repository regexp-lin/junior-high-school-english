# 中考英语作文生成

## 作文题目（情境+内容要求+注意事项）

每个人都会经历失败。假设你是李华，请以"A Lesson from Failure"为题写一篇短文，内容包括：

① 描述一次你失败的经历  
② 你当时的感受  
③ 你从中学到了什么  

**注意：** 词数 80-100；可适当发挥；不得出现真实姓名和校名。

---

### 题目信息

| 项目 | 内容 |
|------|------|
| 体裁 | 话题作文/叙事+感悟类 |
| 话题 | 从失败中学习（A Lesson from Failure） |
| 人称 | 第一人称 |
| 时态 | 一般过去时（叙事）+ 一般现在时（感悟） |
| 字数 | 80-100词 |
| 要点数 | 3个 |

---

## 写作思考

### 审题

- **情境**：分享一次失败经历和从中的收获
- **身份**：初中学生李华
- **核心任务**：叙述失败 → 描述感受 → 总结教训
- **限制**：80-100词，不出现真实姓名校名

### 结构安排

| 段落 | 内容 | 预计词数 |
|------|------|----------|
| 开头 | 引出话题 | 10-12词 |
| 第一段 | 失败经历 | 30-35词 |
| 第二段 | 当时感受 | 15-20词 |
| 第三段 | 学到什么 | 25-30词 |
| 结尾 | 感悟 | 10-12词 |

### 表达准备（三要点中英文）

| 要点 | 中文 | 英文表达参考 |
|------|------|--------------|
| ① 失败经历 | 数学考试不及格、比赛输了 | failed the maths exam; lost the competition |
| ② 感受 | 失望、沮丧、想放弃 | disappointed; frustrated; wanted to give up |
| ③ 教训 | 失败是成功之母、要学会反思和坚持 | failure is the mother of success; learn to reflect and keep trying |

### 可加分亮点（4条）

1. 用 **Nobody is perfect, and everyone faces failure** 开篇
2. 用 **I felt as if the whole world had fallen apart** 夸张表达感受
3. 用 **It was then that I realized...** 强调句引出领悟
4. 用 **Failure doesn't define us — how we respond does** 结尾金句

## 参考范文

**A Lesson from Failure**

Nobody is perfect, and I'd like to share a failure that taught me an important lesson.

Last semester, I spent weeks preparing for the school maths competition. I was confident I could win a prize. However, when the results came out, I didn't even make it to the top thirty. I felt as if the whole world had fallen apart. I was so disappointed that I didn't want to talk to anyone.

My maths teacher noticed my sadness and told me, "Don't be afraid of failure. It shows you where you need to improve." It was then that I realized failure is not the end — it's a new beginning. I started reviewing my mistakes carefully and practicing harder.

Now I understand that failure doesn't define us — how we respond to it does.

（约125词）

---

## 范文亮点分析

### 内容方面

- 三要点完整：① 数学竞赛失利 ② 感到世界崩塌般失望 ③ 老师开导，认识到失败是新的开始
- 情感起伏明显，转折自然

### 结构方面

- 开头引出失败话题
- 按时间线叙述（准备 → 失败 → 失落 → 领悟）
- 结尾金句有力

### 语言方面

- **felt as if the whole world had fallen apart** — 感觉整个世界都崩塌了
- **so... that...** — 如此……以至于……
- **It was then that I realized** — 正是那时我意识到（强调句）
- **reviewing my mistakes** — 复盘我的错误
- **Failure doesn't define us** — 失败不能定义我们

## 全文翻译

**从失败中学到的一课**

没有人是完美的，我想分享一次教会了我重要一课的失败。

上学期，我花了好几周准备学校数学竞赛。我自信满满，觉得一定能获奖。然而成绩出来时，我连前三十名都没进。我感觉整个世界都崩塌了。我太失望了，不想和任何人说话。

数学老师注意到了我的沮丧，对我说："不要害怕失败。它让你知道需要改进的地方。"正是那时我意识到，失败不是终点——而是一个新的开始。我开始认真复盘错误，更加努力地练习。

现在我明白了，失败不能定义我们——我们如何应对它才能。

## 可继续补充

### 好词好句（5行表格）

| 表达 | 释义 | 例句 |
|------|------|------|
| as if the world had fallen apart | 好像世界崩塌了 | I felt as if the whole world had fallen apart. |
| so... that... | 如此……以至于…… | I was so disappointed that I didn't talk to anyone. |
| It was then that I realized | 正是那时我意识到 | It was then that I realized failure is not the end. |
| reviewing my mistakes | 复盘我的错误 | I started reviewing my mistakes carefully. |
| doesn't define us | 不能定义我们 | Failure doesn't define us. |

### 易错点（5个 ✅❌对比）

| 易错点 | ❌ 错误 | ✅ 正确 |
|--------|--------|--------|
| as if 虚拟语气 | felt as if the world falls | felt as if the world had fallen apart（虚拟用过去完成时） |
| make it | didn't make it into | didn't make it to the top thirty（make it to 到达） |
| It was that 强调句 | It was then I realized | It was then that I realized（强调句不能省 that） |
| respond to 搭配 | respond failure | respond to failure（respond to 固定搭配） |
| so that 区分 | so disappointed so that | so disappointed that（so...that 不加 so） |

### 升格建议（3条）

| 建议 | 普通表达 | 升格表达 |
|------|----------|----------|
| 1. 用 as if 夸张表达感受 | I was very sad. | I felt as if the whole world had fallen apart. |
| 2. 用强调句引出领悟 | Then I knew failure was OK. | It was then that I realized failure is not the end. |
| 3. 用金句结尾 | We should not be afraid of failure. | Failure doesn't define us — how we respond to it does. |
